살아있는 숲 V1.58 test — 첫 방문 UX/UI 개편판

목표
- 블로그/검색으로 처음 들어온 사용자가 혼자서 무엇을 해야 하는지 이해하도록 첫 화면을 정리했습니다.

변경 사항
1. 첫 화면 상단에 “처음 온 분을 위한 시작 안내” 카드 추가
   - 오늘의 기록 시작하기 버튼을 가장 큰 행동으로 배치
   - 나무 이름 정하기 → 마음 고르기 → 변화 보기 3단계로 간단히 안내

2. 30초 온보딩 모달 추가
   - 첫 방문 로컬 사용자에게 자동 표시
   - 안내 닫기/내 숲 시작하기 제공
   - 초대 링크로 들어온 사용자 흐름은 기존 초대 카드 우선 유지

3. 가독성/타겟 감성 개선
   - 어두운 배경 중심에서 파스텔 숲 카드 톤으로 보정
   - 글씨 대비, 버튼 대비, 카드 배경 밝기 개선
   - 모바일에서 시작 버튼과 안내 카드가 세로로 자연스럽게 보이도록 조정

4. 정원 진입 후 첫 기록 안내 추가
   - 나무 이름 → 마음 선택 → 변화 확인 순서를 다시 보여주는 짧은 안내 카드 추가

유지 사항
- 저장키 livingForestV012 유지
- 기존 카카오 초대/친구 자리/방문자/관리 기능 유지
