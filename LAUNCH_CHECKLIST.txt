살아있는 숲 V1.70.3 test 확인 체크리스트

[기본]
- 화면 상단 버전이 V1.70.3 test로 보인다.
- 월드 숲과 내 정원 기본 이동이 기존처럼 작동한다.
- GitHub Pages 배포 후 Ctrl + F5 새로고침해도 화면이 깨지지 않는다.

[전체숲]
- 전체숲이 비어 보이지 않고 여러 나무가 모인 숲처럼 보인다.
- 초반에는 더미 나무가 숲을 채운다.
- 더미 나무에는 이름 라벨이 보이지 않는다.
- 실제 친구가 들어오면 가까운 더미 자리가 실제 친구 나무로 바뀌는 방향이 유지된다.
- 실제 친구 나무에는 이름/라벨이 보인다.

[내 나무 흐름]
- 처음 온 사람은 내 나무 이름을 지을 수 있다.
- 오늘 마음을 고르면 내 나무 기록이 남는다.
- 기록 후 내 나무 상태와 성장 정보가 갱신된다.

[정원 꾸미기]
- 들꽃 버튼을 누르면 선택 상태가 표시된다.
- 조약돌 버튼을 누르면 선택 상태가 표시된다.
- 작은 등불 버튼을 누르면 선택 상태가 표시된다.
- 선택한 표식이 내 정원 무대에 표시된다.

[친구 초대/자리]

- 자리 이동/교체 후 친구 목록이 유지된다.
