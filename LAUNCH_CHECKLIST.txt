살아있는 숲 V1.46 test 확인 체크리스트

[버전 / 기본]
- 화면 상단 버전이 V1.46 test로 표시된다.
- 월드 숲 / 내 정원 / 감정 기록 / 하단 패널이 기존처럼 이동된다.

[방문자]
- 정원 무대의 새 방문자가 귀여운 v2 이미지로 보인다.
- 정원 무대의 다람쥐 방문자가 귀여운 v2 이미지로 보인다.
- 월드 숲 작은 생명 표시의 새 / 다람쥐도 v2 이미지로 보인다.
- 방문자 등장 / 기록 / 흔적 카드가 기존처럼 작동한다.

[꾸미기]
- 꾸미기 탭의 정원 장식 카드가 리본 / 가랜드 / 꽃 랜턴 / 소풍 세트 4가지로 보인다.
- 각 버튼에 이미지 썸네일이 보인다.
- 장식을 누르면 선택 상태가 바뀐다.
- 장식을 선택하면 정원 무대 위에도 실제 이미지로 보인다.
- 새로고침 후에도 선택한 장식이 유지된다.

[기타]
- 보관 파일 받기 / 불러오기 기능이 작동한다.
- 관리자 화면이 기존처럼 열린다.
