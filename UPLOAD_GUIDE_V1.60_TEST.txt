V1.60 test 업로드 안내

1. ZIP 압축을 풀고 전체 파일을 GitHub Pages 저장소에 덮어쓰기 업로드합니다.
2. index.html, app.js, style.css가 반드시 같이 교체되어야 합니다.
3. 업로드 후 브라우저에서 강력 새로고침(Ctrl+F5) 또는 시크릿 모드로 확인합니다.
4. 확인 순서
   - 상단 버전이 V1.60 test인지 확인
   - 첫 방문 안내창에서 “내 숲 시작하기” 클릭
   - 정원 화면에서 기록 패널이 자동으로 열리는지 확인
   - 나무 이름 입력칸과 마음 선택 버튼이 숲 이미지보다 먼저 보이는지 확인
   - 이름 저장 및 마음 선택이 정상 동작하는지 확인
