# CHANGES V1.57 TEST

## 목표

V1.56 test에서 정리한 친구 배치 UX를 기준으로, 실제 사용 중 실수하기 쉬운 리배치/교체 흐름을 안정화했습니다.

## 변경 사항

1. 중복 클릭 방지
- 친구 자리 저장 중에는 다른 자리 버튼을 다시 누르지 못하게 했습니다.
- 클릭 이벤트에서도 저장 중/비활성 버튼은 무시합니다.

2. 현재 자리 잠금
- 이미 그 친구가 있는 자리는 다시 저장하지 않도록 버튼을 잠금 처리했습니다.
- 사용자는 현재 자리와 선택 가능한 자리를 더 쉽게 구분할 수 있습니다.

3. 이동/교체 상태 구분
- 빈 자리에 처음 배치: 빈 자리
- 이미 다른 자리에 있는 친구를 빈 자리로 옮김: 이동
- 다른 친구가 있는 자리로 옮김: 교체

4. 교체 확인창 개선
- 기존 자리와 새 자리를 함께 보여줍니다.
- 교체된 친구는 친구 목록에서 삭제되지 않는다는 안내를 유지했습니다.

5. 취소 안내 추가
- 교체 확인창에서 취소를 누르면 변화가 없다는 안내를 표시합니다.

## Apps Script

이번 버전은 프론트 안정화 패치입니다.
Apps Script 새 배포는 필요 없습니다.
