살아있는 숲 V1.41 test 변경 요약

목적:
- V1.40의 나무 중심 감성 UX/UI 전환을 안정화합니다.
- 새 대형 기능을 추가하기보다, 하단 패널이 실제로 안전하게 작동하도록 정리합니다.

1. 불필요한 부분 제거
- 현재 index.html에 존재하지 않는 예전 진행 흐름 카드(flowTitle / flowDescription / flowStep*) 관련 JS 코드를 제거했습니다.
- 사용하지 않는 service-flow / flow-steps CSS를 제거했습니다.
- 이전 패치 안내 TXT는 실행에 필요하지 않으므로 clean upload ZIP에서는 제외했습니다.

2. 틀린 부분 수정
- V1.40 app.js에서 gardenHubElement, gardenHubSheetElement, gardenPanelTitleElement, closeGardenPanelBtnElement, gardenHubTabButtons가 선언되지 않아 앱 실행 중 멈출 수 있던 문제를 수정했습니다.
- index.html / style.css / app.js 캐시 버전을 V1.41 test로 맞췄습니다.
- README / VERSION / LAUNCH_CHECKLIST / UPLOAD_GUIDE의 설명을 V1.41 기준으로 정리했습니다.
- assets 폴더 포함 업로드 기준으로 안내 문구를 수정했습니다.

3. 부족한 부분 추가
- 하단 패널 탭 클릭 이벤트가 중복 연결되지 않도록 gardenHubLayoutBuilt 상태값을 추가했습니다.
- 잘못된 탭 키가 들어와도 record 탭으로 안전하게 돌아가도록 방어 코드를 추가했습니다.
- 내 정원 진입 시 하단 패널 레이아웃을 보장하고, 접힌 상태면 마지막 탭을 다시 열도록 보강했습니다.
- 하단 패널 탭에 aria-controls를 추가해 탭과 패널 연결성을 보강했습니다.
- 하단 패널 focus-visible 스타일을 추가해 키보드 접근성을 보강했습니다.

유지한 부분:
- 관리자 / GA4 / Google Sheets / Apps Script 구조는 건드리지 않았습니다.
- localStorage 저장 키와 schema 12 구조를 유지했습니다.
- 방문자 시스템, 감정 기록, 나무 이미지, assets 경로를 유지했습니다.
