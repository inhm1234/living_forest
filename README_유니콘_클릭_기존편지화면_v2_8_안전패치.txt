유니콘 클릭 → 기존 편지 작성 화면 v2.8 안전 패치

업로드 파일
- dev/garden-login-test/garden.js
- dev/garden-login-test/forest-unicorn-preview.js

이번 패치의 범위
1) ?forestFriendPreview=1에서 유니콘을 누르면, 기존 #letterComposerSheet를 유니콘 전용으로 엽니다.
2) 받는 친구·제목·본문은 기존 편지 작성 화면에서 그대로 입력합니다.
3) ‘유니콘에게 편지 맡기기’를 누르면 실제 DB/RPC는 호출하지 않고, 유니콘 출발·8초 뒤 귀환만 재생합니다.
4) 작성창을 X나 바깥 영역으로 닫으면 유니콘은 다시 평소 루틴으로 돌아갑니다.
5) 하단 ‘편지’의 기존 동물 배송·받은 편지·배송 중 목록은 수정하지 않습니다.

안전 장치
- 일반 방문 동물의 renderLetterComposer / sendGardenLetter / RPC 본문은 바꾸지 않았습니다.
- 유니콘 체험일 때만 sendGardenLetter 첫 부분에서 별도 미리보기 분기로 빠집니다.
- 일반 동물 편지 작성과 전환될 때 특별 친구 상태를 자동 해제합니다.

검수 순서
1. Ctrl+F5 후 기존 정원과 하단 버튼이 정상인지 먼저 확인
2. 유니콘 클릭 → ‘숲 유니콘에게 편지를 맡기기’ 작성창이 열리는지 확인
3. 작성창 X → 유니콘이 다시 정원 루틴으로 돌아오는지 확인
4. 받는 친구/제목/본문 입력 → ‘유니콘에게 편지 맡기기’ → 출발/사라짐/귀환 확인
5. 하단 ‘편지’ 버튼을 열어 기존 목록과 버튼이 그대로인지 확인
