살아있는 숲 V1.27.1 test 숲의 소리 버튼 수정 안내

수정 이유:
- V1.27에서 숲의 소리 카드와 Web Audio 함수는 들어갔지만, 버튼 클릭 이벤트 연결이 빠져 있었습니다.
- 그래서 바람 / 물방울 / 밤벌레 / 끄기 버튼을 눌러도 실제로 동작하지 않을 수 있었습니다.

수정 내용:
- data-forest-sound 버튼 클릭 시 playForestSound()가 실행되도록 연결
- 끄기 버튼 클릭 시 stopForestSound()가 실행되도록 연결
- 전체 숲으로 돌아가거나 화면을 숨기면 숲의 소리가 멈추도록 처리
- 버전 표시와 캐시값을 V1.27.1 test로 갱신

확인 방법:
1. 일반 화면 상단이 V1.27.1 test인지 확인
2. 내 정원으로 이동
3. 숲의 소리 카드에서 바람 / 물방울 / 밤벌레 버튼 클릭
4. 소리가 나고 선택 상태가 바뀌는지 확인
5. 끄기 버튼으로 멈추는지 확인
