DEV v0.3.2 — 좌표 맵·안전 카메라 보정

v0.3.1 화면의 빈 하늘색 영역/맵 밀림을 수정한 패치입니다.

수정 내용
- 배경 원본 크기 1448×1086과 월드 좌표계를 일치시킴
- 카메라 확대값을 화면 크기에 맞춰 자동 보정함
- 카메라 이동값을 월드 내부 범위로 제한함
- 따라서 화면 가장자리에 빈 배경이 드러나지 않음
- 기존 저장 데이터의 예전 슬롯 ID도 새 고정 구역으로 자동 보정함
- 아무도 배정되지 않은 일반 슬롯 표시는 숨기고, 내 자리와 미래 예약 자리만 보이게 함

업로드
- 이 ZIP을 풀고 dev 폴더 안의 내용물만 GitHub 저장소의 기존 dev 폴더에 덮어쓰기
- /dev/ 주소에서 Ctrl + F5로 새로고침
