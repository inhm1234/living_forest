오늘의숲 개발용 앱 폴더입니다.

이 폴더는 service-v0.1-stable 안정 버전을 복사해서 만든 개발 테스트 버전입니다.

접속 주소 예시:
https://inhm1234.github.io/living_forest/dev/

운영 규칙:
1. 실제 서비스는 저장소 루트(index.html)를 사용합니다.
2. 새 기능, 광고, 이미지 교체는 먼저 /dev/에서 테스트합니다.
3. 문제가 없을 때만 루트 서비스 파일에 반영합니다.
4. dev 버전은 localStorage 저장키가 _DEV로 분리되어 서비스 데이터와 섞이지 않습니다.


주의: 이 개발 버전은 /dev/assets 폴더를 따로 복사하지 않고, 상위 폴더의 ../assets 이미지를 함께 사용합니다. 그래서 업로드 용량이 작고, 실제 서비스 이미지 파일을 중복하지 않습니다.
