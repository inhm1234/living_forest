오늘의숲 · 내 정원 프로토타입 v0.1

목적
- 큰 숲 배경 대신, 내 나무를 중심으로 기록·동물·날씨·편지를 경험하는 흐름을 확인하는 개발용 화면입니다.
- 기록과 편지는 브라우저 localStorage에만 저장됩니다. 실제 친구 전송 기능은 아닙니다.

동작 확인 순서
1. '마음 남기기'를 눌러 감정과 한 줄을 적고 저장합니다.
2. '내 기록'에서 저장한 글이 보이는지 확인합니다.
3. 나뭇가지의 편지를 눌러 읽습니다.
4. 편지를 읽은 뒤 '편지'에서 보관되는지 확인합니다.
5. 오른쪽 위 날씨 버튼을 눌러 바람 / 비 / 햇살 변화를 확인합니다.
6. 아래 '테스트 상태 처음으로'를 누르면 처음 상태로 돌아갑니다.
