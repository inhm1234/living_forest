오늘의숲 · 개인 화면 시야 검증 v0.2

목적
- 선택한 내 나무는 화면 중앙에서 충분히 보여야 한다.
- 나무, 큰 바위, 능선처럼 내 나무를 가릴 수 있는 요소를 구분해 처리한다.

이번 DEV 규칙
1) 앞쪽 중앙 시야 통로의 다른 나무: 개인 화면에서만 숨김
2) 앞쪽 중앙 시야 통로의 큰 바위: 좌표는 유지하고 약하게 표시
3) 능선·절벽 같은 큰 지형: 지우지 않고 카메라 높이/각도를 보정
4) 개울·길 등 낮은 요소: 그대로 표시

실제 월드 01로 갈 때
- 현재 테스트의 간단한 화면 판정은, 실제 지형 폴리곤/높이 데이터 기반 판정으로 확장한다.
- 카메라 보정으로도 내 나무가 충분히 안 보이는 자리는 애초에 나무 슬롯으로 쓰지 않는다.
