오늘의숲 /dev/world-layout/ — 공동 숲 카메라 검증 v0.1

이 폴더는 기존 DEV 서비스 화면을 수정하지 않는 별도 구조 실험장입니다.

접속 주소:
https://inhm1234.github.io/living_forest/dev/world-layout/

이번 파일의 목적:
- 16개 고정 슬롯의 x, y, z 좌표 테스트 (4개씩 4개의 작은 숲 군락)
- 슬롯마다 선택 시 같은 월드에서 선택한 나무만 화면 중앙에 오는지 확인
- 다른 나무가 앞·뒤 배경처럼 그려지는지 확인
- 북/동/남/서 방향을 바꿔도 좌표는 같고 카메라만 달라지는지 확인

중요:
- 이 버전은 최종 서비스 화면이 아닙니다.
- 최종 나무 PNG / 감정 기록 UI / 기존 정원 배경은 사용하지 않습니다.
- 나무 형태는 카메라 깊이와 겹침을 검증하는 단순한 임시 표시입니다.
- 16개의 나무는 모두 '이미 심겨 있는 사용자 나무'라고 가정합니다.
