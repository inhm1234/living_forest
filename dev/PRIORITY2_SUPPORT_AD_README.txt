오늘의숲 DEV v0.2.1 - 수익화 테스트

변경 내용
1. 메인 화면 하단에 후원 안내 버튼을 추가했습니다.
2. 후원 버튼을 누르면 결제창이 아니라 후원 안내 팝업만 열립니다.
3. 작은 광고 테스트 영역을 추가했습니다.
4. 실제 광고 코드는 아직 삽입하지 않았습니다.
5. 이 변경은 dev 버전에서 먼저 확인한 뒤 서비스 버전에 반영해야 합니다.

검수 기준
- 나무와 정원을 가리지 않는지 확인
- 모바일 화면에서 답답하지 않은지 확인
- 후원 버튼을 눌렀을 때 팝업이 열리고 닫히는지 확인
- 광고 영역이 너무 크거나 분위기를 해치지 않는지 확인
