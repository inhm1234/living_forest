# DEV v0.2.7 · 성장 규칙 정리 / 정원-전체숲 단계 동기화

이번 버전은 광고 신청과 서비스 운영을 위한 기본 안내 페이지를 준비한 dev 테스트 버전입니다.

## 변경 내용

- `/dev/privacy.html` 추가
- 전체숲 하단 운영 도움 패널에 `개인정보·광고 안내` 링크 추가
- 하단 문서 링크에 `개인정보·광고 안내` 추가
- 앱 버전 표시를 `DEV v0.2.7 · 성장 규칙 정리 / 정원-전체숲 단계 동기화`로 변경

## 유지한 원칙

- 실제 광고 코드는 아직 넣지 않음
- 실제 후원 링크나 계좌번호는 아직 넣지 않음
- 내 정원에서는 수익화 패널을 계속 숨김
- 광고/후원은 전체숲 하단에서만 조용히 테스트
