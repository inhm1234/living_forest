오늘의숲 — 인벤토리 v3.2 보관함 서랍 열기 수정

수정 파일
- index.html
- garden.js

문제
- 꾸미기 중 "🧺 보관함 N" 버튼을 눌러도 서랍이 열리지 않음.

원인
- JavaScript는 hidden 속성만 해제했지만, HTML에 남아 있는 .hidden 클래스가 display:none을 계속 적용하고 있었음.

수정
- 서랍을 열고 닫을 때 hidden 속성과 .hidden 클래스를 같이 변경하도록 고침.
- garden.js 주소에 버전값을 바꿔, 배포 후 이전 JS 캐시가 남지 않도록 함.

확인 순서
1. Ctrl + Shift + R로 새로고침
2. 꾸미기
3. 정원 장식 하나를 보관함에 넣기
4. 정원 안 하단의 "🧺 보관함 N" 누르기
5. 작은 것 보관함 서랍이 열리고, 장식 카드를 누르면 정원에 다시 나타나는지 확인
