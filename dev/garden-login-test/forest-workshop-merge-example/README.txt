[체험판] 오늘의숲 · 작은 풍경 만들기

목적
- 마음 기록과 나무 성장을 끝낸 뒤에도, 사용자가 내 정원의 빈자리를 완성하려고 5~10분 더 머물 수 있는지 확인하는 독립 체험판입니다.
- 실제 정원/로그인/Supabase/기존 장식/편지/동물/PWA 파일은 건드리지 않습니다.

체험 흐름
1. 같은 재료 두 개를 차례로 눌러 합칩니다.
2. 씨앗→새싹→꽃무리 / 이끼 조각→폭신한 이끼→이끼 돌 / 반짝 조각→작은 빛→빛병을 만듭니다.
3. 완성된 물건을 '고슴도치의 작은 쉼터'에 놓습니다.
4. 3개를 모두 놓으면 정원에 남는 작은 풍경이 완성됩니다.

실서비스 가설(이번 체험판에는 연결하지 않음)
- 마음 기록 뒤 즉시 작업 1회
- 이후 4시간마다 작업 1회 누적
- 최대 3회 보관
- 프로젝트는 3~5일 정도에 하나를 완성
