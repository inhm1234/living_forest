오늘의숲 · 바람길 편지 배달 v1 (독립 체험판)

목적
- 기존 정원, 로그인, 기록, 편지, 동물 배송, 장식, PWA 코드를 수정하지 않습니다.
- 30초짜리 실제 플레이 감각을 검수하기 위한 별도 게임 프로토타입입니다.

업로드 위치
- GitHub 저장소 안의 아래 새 폴더 전체를 올립니다.
  dev/garden-login-test/letter-wind-game-example/

체험 주소
- https://inhm1234.github.io/living_forest/dev/garden-login-test/letter-wind-game-example/

플레이
- 분홍 봉투: 점수 +100
- 별편지: 점수 +250, 콤보 +2
- 낙엽/빈 종이: 콤보만 끊기며 점수는 깎이지 않습니다.
- 10초씩 3개의 바람길, 총 약 30초입니다.
- 최고 점수는 실제 DB가 아니라 현재 브라우저 localStorage에만 저장됩니다.

이번 검수에서 볼 것
1. 봉투를 움직여 받는 조작이 직관적인가?
2. 30초 동안 목표감과 한 판 더 하고 싶은 감각이 생기는가?
3. 낙엽을 피하고 콤보를 잇는 판단이 너무 쉽거나 너무 피곤하지 않은가?
4. 오늘의숲 세계관 안에서 편지·바람·정원과 어색하지 않은가?

주의
- 이 폴더는 독립 체험판입니다. 실제 정원에 붙이거나 DB를 건드리지 않습니다.
