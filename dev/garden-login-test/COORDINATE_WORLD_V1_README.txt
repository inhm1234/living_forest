DEV 검수 방법
1. 모바일에서 장식 하나를 나무 뿌리 가까이에 옮기고 ‘배치 저장’을 누름
2. PC에서 같은 계정·같은 DEV 주소를 열어 나무와 장식의 상대 위치를 확인
3. PC에서 다시 조금 옮겨 저장한 뒤 모바일에서 같은 상대 위치인지 재확인
4. 방문 동물, 받은 편지 봉투, 비·바람 화면이 겹치거나 클릭되지 않는지 함께 확인

알아둘 점
- 기존에 저장해 둔 장식 좌표는 값 자체를 삭제하거나 변환하지 않음
- 기존 좌표는 이전 화면 전체 기준으로 저장되어 있었으므로, 첫 DEV 확인에서 한 번만 조금 다르게 보일 수 있음
- 새로 저장한 배치부터는 공통 정원 좌표 기준으로 유지되는 것이 목표임
