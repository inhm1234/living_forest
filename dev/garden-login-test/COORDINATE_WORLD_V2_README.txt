오늘의숲 DEV - 공통 정원 좌표 v2

수정 범위
- index.html: garden.css / garden.js에 v2 캐시 버전 문자열을 붙였습니다.
- garden.css: 꾸미기 중 touch-action과 선택 방지를 장식 레이어와 이미지에 명시했습니다.
- garden.js: 장식을 잡은 뒤 pointermove / pointerup을 window 캡처 단계에서 추적하고, 레이어 포인터 캡처를 사용합니다.

검수 순서
1. 모바일 DEV에서 꾸미기 > 장식 하나를 잡아 움직이고 배치 저장
2. PC DEV를 새로고침하여 같은 장식이 나무 기준 같은 자리에 있는지 확인
3. PC에서 다른 장식을 옮겨 저장 후 모바일에서 같은 자리인지 다시 확인

중요
- 기존 position_x / position_y는 예전 화면 전체 기준으로 저장된 값입니다.
- 공통 정원 좌표로 전환하는 첫 저장은, 현재 가장 자연스럽다고 느끼는 기기(권장: 모바일)에서 장식을 한 번씩 확인·조정한 뒤 저장해 주세요.
- 이번 파일은 /dev/garden-login-test/ 전용입니다. 운영 루트에는 적용하지 않습니다.
