오늘의숲 인벤토리 v3.1 — 클릭/안내 수정

수정 이유
- 정원 안의 "보관함에 넣기" 버튼이 garden-world의 pointer-events: none 영향을 받아 실제로 클릭되지 않았습니다.
- 꾸미기 중 보관함 버튼이 아이콘+숫자만 보여, 넣어둔 장식을 어디서 꺼내는지 알기 어려웠습니다.

수정 내용
1. 정원 안 "보관함에 넣기" 버튼을 클릭 가능하도록 수정했습니다.
2. 꾸미기 중 버튼을 "🧺 보관함 1"처럼 보이게 했습니다. (작은 모바일 화면에서는 공간 때문에 글자만 숨김)
3. 장식을 보관함에 넣으면 작은 서랍이 자동으로 열려, 방금 넣은 장식을 바로 확인하고 다시 꺼낼 수 있습니다.
4. 안내 문구를 "한 번 눌러 보관함에 넣기"로 명확하게 바꿨습니다.

Supabase SQL은 다시 실행할 필요 없습니다.
