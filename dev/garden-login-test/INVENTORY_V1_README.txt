인벤토리 v1 적용 안내

이번 작업 범위
- 오늘 기록 후 찾은 자연물은 바로 정원에 강제 배치되지 않고 ‘작은 것 보관함’으로 들어갑니다.
- 보관함에서 ‘정원에 놓기’를 누르면 기본 자리에 나타납니다.
- 정원에 놓은 자연물은 보관함에서 다시 회수할 수 있습니다.
- 보관했다가 다시 꺼내면, 마지막으로 두었던 자리를 그대로 기억합니다.
- 기존에 정원에 놓여 있던 자연물은 그대로 유지됩니다.
- 친구 정원에는 실제로 정원에 꺼내놓은 자연물만 보입니다.
- 선물/동물 배송/희귀도/교환은 이번 범위에 포함하지 않습니다.

검수 순서
1) 기존 정원 장식이 그대로 보이는지 확인
2) 오늘 기록 후 반짝임을 눌러 새 장식을 찾기
3) 새 장식이 정원이 아니라 ‘보관함’에 들어갔는지 확인
4) 보관함에서 ‘정원에 놓기’ → 정원에 표시되는지 확인
5) 보관함의 ‘정원에 놓인 작은 것’에서 ‘보관함에 넣기’ → 정원에서 사라지고 보관 중 목록으로 돌아오는지 확인
6) 정원에 꺼낸 뒤 ‘꾸미기’에서 드래그하고 저장 → 새로고침 뒤 위치가 유지되는지 확인

주의
- SQL을 적용하기 전에는 화면은 열리지만 보관함에 넣기/꺼내기가 저장되지 않습니다.
- 이 v1은 DEV 경로만 수정합니다. 운영 루트는 건드리지 않습니다.
