오늘의숲 · 인벤토리 v2 (정원을 보며 꾸미기)

이번 수정
- 보관함을 큰 바텀시트로 열지 않습니다.
- 보관함/꾸미기 버튼을 누르면 정원은 그대로 보이고, 정원 아래에 작은 보관함이 열립니다.
- 보관 중 장식을 누르면 정원에 꺼낼 수 있습니다.
- 정원에 놓인 장식을 톡 누르면 선택되고, 아래 보관함에서 다시 넣을 수 있습니다.
- 장식 위치 변경은 기존처럼 ‘배치 저장’에서 저장합니다.
- 보관함에 넣기/정원에 놓기는 기존 inventory / placed 상태 저장을 그대로 사용합니다.
