# DEV v0.2.7 · 성장 규칙 정리 / 정원-전체숲 단계 동기화

이번 수정 내용
- 나무는 시간이 아니라 감정 기록 횟수에 따라 성장하도록 기준을 정리했습니다.
- 내 정원과 전체숲에서 내 나무가 같은 성장 단계로 보이도록 수정했습니다.
- 전체숲에서 내 나무를 과도하게 큰 단계로 보정하던 로직을 제거했습니다.
- 전체숲에서는 성장 단계는 그대로 두고, 크기/위치만 CSS에서 보이도록 유지합니다.

테스트 포인트
1. 새로 시작한 나무가 내 정원과 전체숲에서 비슷한 단계로 보이는지
2. 1~2회 기록 시 새싹 단계로 보이는지
3. 3~6회 기록 시 작은 나무 단계로 보이는지
4. 7회 이상 기록 시 자리 잡은 나무 단계로 보이는지
5. 전체숲에서 내 나무가 이전처럼 과하게 커 보이지 않는지
