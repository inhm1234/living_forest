# 오늘의숲 DEV v0.3.2 — 좌표 기반 2.5D 월드

## 이번 프로토타입에서 바뀐 구조

`world-data.js`가 하나의 전체 월드를 정의합니다.

- 월드 크기: `3200 × 1800`
- 작은 숲 구역: 8개
- 나무 슬롯: 32개
- 사용자당 예약 슬롯: 4개
- 카메라: 사용자의 작은 숲 구역을 중심으로 이동

각 슬롯은 다음 데이터를 가집니다.

```js
{
  id: "rainbow-01",
  groveId: "rainbow-path",
  x: 1460,
  y: 1000,
  z: 12,
  scale: 0.97,
  role: "current"
}
```

- `x`, `y`: 전체 월드에서의 지면 좌표
- `z`: 언덕·단차 높이. 렌더링할 때 화면 위쪽으로 들어 올립니다.
- 앞뒤 순서: 투명도 대신 **나무 밑동의 y값**으로 자동 정렬합니다.

## 현재 DEV 저장 데이터

각 기기에 저장되는 나무 데이터에 아래 필드가 추가됩니다.

```js
worldGroveId
worldSlotId
reservedWorldSlotIds
```

기존 사용자는 첫 실행 시에도 자동으로 한 구역과 미래 슬롯 3개를 배정받습니다.

## 지금 보이는 것

- 분홍색 테두리: 내 현재 나무의 고정 자리
- `다음`: 같은 구역 안에 예약된 다음 나무 자리
- 얇은 점선: 아직 비어 있는 전체 월드의 배정 가능 자리
- 나무는 내 나무와 실제로 연결된 온라인 친구 나무만 표시합니다. 더미 나무는 새 월드 뷰에서 사용하지 않습니다.
