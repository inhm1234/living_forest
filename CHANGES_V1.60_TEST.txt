살아있는 숲 V1.60 test 변경점

목표
- 온보딩에서 “내 숲 시작하기”를 누른 뒤 바로 무엇을 해야 하는지 보이게 만드는 첫 기록 동선 개선.
- 검색/블로그로 처음 들어온 사람이 숲 이미지 구경 전에 나무 이름 입력과 마음 선택을 바로 찾도록 정리.

변경 내용
1. 정원 화면에서 내 정원 메뉴/기록 패널 위치를 숲 이미지 위쪽으로 이동
   - 시작 직후 입력해야 할 영역이 먼저 보이도록 구성했습니다.

2. 오늘의 기록 시작하기 / 내 숲 시작하기 클릭 시 기록 패널 자동 열림
   - 나무 이름 입력칸과 마음 선택 버튼이 바로 보이게 했습니다.

3. 첫 기록 위치로 자동 스크롤 및 입력칸 포커스 보완
   - 나무 이름이 없으면 이름 입력칸으로 이동합니다.
   - 이미 이름이 있으면 마음 선택 카드로 이동합니다.

4. 첫 기록 패널 강조 효과 추가
   - 시작 직후 사용자가 “여기서 하면 되는구나”를 느끼도록 부드러운 강조 효과를 넣었습니다.

5. 가독성 유지
   - 기록 패널, 입력칸, 마음 선택 버튼의 크기와 대비를 조금 더 올렸습니다.

확인할 것
- 첫 방문 온보딩에서 “내 숲 시작하기” 클릭 후 기록 패널이 열리는지
- 나무 이름 입력칸이 바로 보이는지
- 입력칸에 커서가 들어가는지
- 마음 선택 버튼이 이름 입력 아래에 자연스럽게 보이는지
- 숲 이미지와 기존 카카오 초대 기능이 유지되는지
