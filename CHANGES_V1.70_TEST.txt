# V1.70 test — 전체 숲 나무 군집 / 더미 나무 대체판

목표:
- 전체 숲을 버튼/메뉴가 아니라 사람들이 만든 나무들이 모인 숲처럼 보이게 한다.
- 초반 유저가 적을 때도 숲이 비어 보이지 않게 더미 나무를 보여준다.
- 실제 친구가 들어오면 가까운 자리의 더미 나무가 실제 친구 나무처럼 보이게 한다.

변경:
1. 전체 숲 렌더링 구조 보강
   - 기존 배경용 나무들은 더미 나무로 취급
   - 친구 자리 5곳은 `worldCommunitySeatSlots`로 따로 잡음
   - 온라인 친구 좌석 데이터가 있으면 해당 자리를 실제 친구 나무로 표시

2. 더미/실제 나무 시각 차이
   - 더미 나무는 살짝 배경처럼 표시
   - 실제 친구 나무는 밝기, 그림자, 이름 라벨을 강화
   - 모바일에서는 라벨이 과하지 않도록 크기 축소

3. 전체 숲 문구 보정
   - “초반에는 더미 나무가 숲을 채우고, 실제 친구가 오면 가까운 자리가 친구 나무로 바뀐다”는 방향으로 정리
   - 큰 월드 기능을 당장 설명하지 않고, 숲이 채워지는 기대감만 남김

4. 출시 기준
   - 지금은 큰 월드/단체 기능 구현이 아니라 빠른 출시 우선
   - 실제 사용자 데이터 확장은 이후 작업으로 남김
