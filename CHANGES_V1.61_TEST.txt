살아있는 숲 V1.61 test — 첫 기록 완료 UX 개선판

핵심 변경
1. 나무 이름 입력 버튼 문구를 "저장"에서 "나무 이름 정하기"로 변경했습니다.
2. 이름을 정한 뒤 마음 선택 카드가 자동으로 강조되고 화면 중앙에 보이도록 했습니다.
3. 마음 선택 잠금/활성 상태 안내 문구를 더 명확하게 바꿨습니다.
4. 아직 잠긴 내일의 씨앗/숲 일기장 카드도 제목과 사용 조건은 읽히도록 가독성을 보정했습니다.
5. 버전 표기를 V1.61 test로 갱신했습니다.

테스트 포인트
- 첫 방문 온보딩에서 내 숲 시작하기 클릭
- 나무 이름 입력 후 "나무 이름 정하기" 버튼 클릭
- 마음 선택 카드가 밝아지고 바로 선택 가능해 보이는지 확인
- 기록 전 아래 잠긴 카드 제목과 조건 문구가 읽히는지 확인
