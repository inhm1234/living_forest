살아있는 숲 V1.59 test 변경점

목표
- V1.58에서 추가한 첫 방문 온보딩을 실제 첫 화면 기준으로 한 번 더 다듬었습니다.
- 지인 테스트가 아니라 블로그/검색 유입으로 들어온 낯선 사용자가 혼자 이해하는 흐름을 기준으로 했습니다.

변경 사항
1. 온보딩 모달 2차 정리
- 배경 blur를 줄여 안내창 뒤의 숲 분위기가 더 보이도록 변경했습니다.
- 첫 방문자가 안내창만 보는 것이 아니라 “여기가 숲 서비스구나”라는 느낌도 받을 수 있게 했습니다.

2. 시작 버튼 강조
- 온보딩의 핵심 버튼과 첫 화면의 “오늘의 기록 시작하기” 버튼을 더 크게/명확하게 정리했습니다.
- 보조 버튼은 투명한 텍스트 버튼에 가깝게 낮춰서 선택지가 너무 많아 보이지 않게 했습니다.

3. 첫 행동 이동 보완
- “내 숲 시작하기/오늘의 기록 시작하기”를 누르면 정원 화면으로 이동한 뒤 나무 이름 입력 위치로 자연스럽게 스크롤됩니다.
- 처음 온 사용자가 정원 화면으로 들어간 뒤 다시 길을 잃지 않도록 보완했습니다.

4. 가독성 유지
- 카드와 입력칸, 안내 카드의 대비를 조금 더 분명하게 조정했습니다.
- 기존의 귀여운 파스텔 숲 톤은 유지했습니다.

유지 사항
- 카카오 초대 기능
- 온라인 친구 자리 기능
- 방문자/기록/보관함/관리 기능
- localStorage 저장키
