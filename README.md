## V1.10.2 test 핵심 변경점

V1.10.2 test는 월드 숲에서 내 나무만 따로 보이는 느낌을 줄이고,
주변 여러 나무가 함께 자라며 숲을 이루는 인상을 더 강하게 주도록 보정한 테스트판입니다.

## 이번 버전에서 확인할 것

- 주변 나무 수와 밀도가 늘어나 숲처럼 느껴지는지
- 가까운 나무와 먼 나무가 크기/선명도 차이로 구분되는지
- 내 나무가 혼자 떠 보이지 않고 주변 나무와 어우러지는지
- 이름표와 설명이 너무 튀지 않고 조용하게 보이는지

## 유지되는 기능

- 월드 숲 / 개인 정원 이동
- 낮 / 노을 / 밤 시간대 연동
- 나무 이름 정하기
- 하루 1번 감정 기록
- 방문자 흔적 / 방문 기록
- 저장 구조 `schema 3` / `local-only`
- 테스트 모드 `?test=1`
