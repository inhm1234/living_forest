# 살아있는 숲 V1.71.0 test

현재 V1.71.0 test는 **정원 메뉴와 실제 기능 화면을 분리한 판**입니다.
정원 화면은 하나의 고정된 무대로 유지하고, 하단 메뉴는 기능 선택만 담당합니다.
`기록 / 활동 / 꾸미기 / 편지 / 보관함`을 누르면 정원 화면을 밀어내지 않고 별도 팝업에서 기능이 열립니다.

## 핵심 방향

- 내 정원/내 숲 무대 크기는 버튼이나 패널 때문에 줄거나 늘어나지 않음
- 접힌 메뉴는 정원 화면 크기에 거의 영향을 주지 않음
- 메뉴를 열어도 실제 기능 내용이 하단 패널 안에서 잘리지 않음
- 실제 기능은 팝업 내부에서만 스크롤됨
- 완료 피드백은 정원 오른쪽 위 작은 로그로 표시
- 전체숲의 더미 나무 + 실제 친구 나무 구조는 유지

## 확인 포인트

1. 상단 버전이 `V1.71.0 test`로 보이는지 확인
2. 정원 첫 화면에서 나무가 크게 보이는지 확인
3. `메뉴 열기`를 눌러도 정원 화면이 아래로 밀리지 않는지 확인
4. `기록 / 활동 / 꾸미기 / 편지 / 보관함`을 누르면 팝업으로 열리는지 확인
5. 팝업 안 내용이 화면 아래에서 잘리지 않고, 팝업 내부에서만 스크롤되는지 확인
6. 전체숲의 더미 나무와 실제 친구 나무 표시가 유지되는지 확인
