# 살아있는 숲 V1.70.8 test

살아있는 숲은 내 나무를 만들고, 오늘의 마음을 남기며, 친구들의 나무가 함께 모여 하나의 숲처럼 보이게 하는 작은 숲 놀이터입니다.

현재 V1.70.8 test는 **접힌 메뉴가 정원 화면 크기에 거의 영향을 주지 않도록 정리한 판**입니다. 버튼을 눌러도 화면이 아래로 밀리지 않도록 접힌 메뉴를 정원 이미지 위 작은 플로팅 버튼으로 바꾸고, 전체숲의 더미 나무 + 실제 친구 나무 구조는 유지합니다.

## 이번 버전 핵심

- 정원 메뉴를 나무를 가리지 않는 하단 안전영역으로 배치
- 버튼/탭을 눌러도 본문 레이아웃이 아래로 밀리지 않고, 내 나무를 덮지 않게 수정
- 정원 무대 높이를 키워 나무가 잘려 보이는 느낌 완화
- 전체숲 배경과 더미 나무/실제 친구 나무 구조 유지
- 내 나무 화면, 친구 초대, 친구 자리 배치, Google Sheets 연동 구조 유지
- 정원 꾸미기 버튼을 `들꽃 / 조약돌 / 작은 등불` 기준으로 정리

## 출시 방향 메모

이 프로젝트의 현재 목표는 지인 홍보형 테스트가 아니라, **블로그 소개글 / 선택적 유튜브 영상 / 검색 유입**으로 처음 온 사람이 혼자 이해하고 첫 행동을 할 수 있는 MVP입니다.

그래서 현재 우선순위는 새 기능 추가보다 다음 기준에 있습니다.

1. 처음 온 사람이 무엇을 해야 하는지 빠르게 알기
2. 전체숲이 비어 보이지 않고 숲처럼 보이기
3. 내 나무 이름 짓기와 오늘 마음 기록까지 자연스럽게 이어지기
4. 친구가 들어왔을 때 숲이 실제 친구 나무로 채워지는 느낌 유지하기
5. 글씨와 버튼이 잘 보이고, 타겟층에 맞는 부드럽고 귀여운 분위기 유지하기

## 업로드 후 확인 체크리스트

1. 상단 버전이 `V1.70.8 test`로 보이는지 확인
2. 전체숲에 여러 더미 나무가 보이는지 확인
3. 더미 나무에는 이름 라벨이 보이지 않는지 확인
4. 실제 친구 나무에는 이름/라벨이 보이는지 확인
5. 내 나무 화면으로 이동했을 때 나무 이름 입력과 오늘 마음 기록 흐름이 유지되는지 확인
6. 정원 메뉴 버튼을 눌러도 화면이 아래로 밀리지 않고, 하단 메뉴영역에서 열리고 나무를 가리지 않는지 확인
7. 정원 꾸미기에서 `들꽃 / 조약돌 / 작은 등불` 버튼이 선택되는지 확인
8. 카카오 초대/친구 자리/Google Sheets 저장 기능이 기존처럼 유지되는지 확인
