## V1.10.14 test 핵심 변경점

V1.10.14 test는 넓고 비어 있는 월드 무대 방향은 유지하되, 화면이 너무 평평하고 허전하게 보이던 문제를 줄이기 위해 길/잔디/풀 기반 바닥감을 추가한 테스트판입니다.

핵심 방향은 다음과 같습니다.
- 넓게 비어 있는 월드 구조는 유지
- 중앙에 길처럼 지나가는 공간과, 그 주변에 잔디/풀/작은 흙자리를 배치
- 나무가 허공에 놓인 것보다 실제로 심겨 있는 느낌이 들도록 접지감 보정
- 비어 있는 자리도 좌석 마커가 아니라 작은 흙자리처럼 보이게 조정
- 멀리 보기 / 가까이 보기 기능은 유지하면서, 가까이 볼수록 길/언덕/풀 결이 조금 더 느껴지도록 보정
- 개인 정원 성장, 나무 성장 이미지, 방문자 흔적, 시간대 연동은 유지

## 이번 버전에서 확인할 것

- 넓게 비어 있어도 화면이 덜 허전하게 느껴지는지
- 길/잔디/풀/작은 흙자리가 자연스럽게 어우러져 보이는지
- 나무가 실제로 심겨 있는 느낌이 나는지
- 중앙 공간이 너무 막힌 숲이 아니라 “지나갈 수 있는 길”처럼 느껴지는지
- 멀리 보기 / 가까이 보기에서 원근감과 바닥감이 조금이라도 살아나는지
