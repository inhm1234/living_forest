# 살아있는 숲

현재 테스트판: V1.41.1 test  
기준 버전: V1.40 test  
최근 업데이트: 하단 패널 가림 보정  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.41.1 test 핵심 변경점

V1.41.1 test는 V1.41에서 작동 안정화한 하단 패널의 기본 펼침 정도를 다시 보정한 버전입니다.
새로운 대형 기능을 늘리기보다, 사용자가 내 정원에 들어왔을 때 먼저 나무와 숲 무대를 보고, 필요할 때만 기록 / 활동 / 꾸미기 / 편지 / 보관함을 펼칠 수 있도록 정리했습니다.

## 수정한 것

- 내 정원 첫 진입 시 패널 본문을 기본 접힘 상태로 변경
- 접힌 상태에서는 메뉴 탭과 패널 열기 버튼만 얇게 표시
- 탭을 누르면 해당 탭 내용이 펼쳐지도록 유지
- 패널 접기 버튼을 패널 열기/접기 토글로 변경
- 펼친 상태의 패널 최대 높이를 줄여 나무와 하단 메시지가 덜 가려지도록 조정
- 정원 무대 높이를 약간 확보해 첫 화면에서 숲 비주얼이 더 잘 보이도록 보정
- V1.41의 하단 패널 실행 안정화 수정 유지
- V1.40 문서에 남아 있던 관리 확장 중심 설명을 V1.41 기준으로 정리
- index.html / app.js / style.css 캐시 버전을 V1.41.1 test로 통일
- assets 폴더 포함 업로드 기준으로 안내 문구 정리

## 유지한 것

- 월드 숲 / 내 정원 / 감정 기록
- 숲 일기장
- 주간 숲 편지
- 오늘의 작은 돌봄
- 정원 표식
- 숲길 산책
- 숲의 소리
- 내일의 씨앗
- 나를 위한 작은 실천
- 내 숲 보관 파일 받기 / 불러오기 / 복원
- 숲 달력 / 내 숲 기억 찾기
- 배지 / 공유 / 초대장 / 방문자 시스템
- 관리자 / GA4 / Google Sheets / Apps Script 구조

## 주의

- 이번 버전은 서버 저장이나 Google 로그인 구현판이 아닙니다.
- 복원 기능은 선택한 보관 파일로 현재 이 기기의 숲 기록을 덮어쓸 수 있습니다.
- 복원 전에는 현재 숲을 먼저 `보관 파일 받기`로 내려받는 것을 추천합니다.
- `assets` 폴더는 포함되어 있으므로 저장소 루트의 `assets/` 구조를 유지해 업로드하세요.
