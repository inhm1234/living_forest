## V1.25 test 핵심 변경점

V1.25 test는 V1.24의 `오늘의 작은 돌봄` 이후에 같은 돌봄 기능을 반복 보강하지 않고, 내 정원 자체를 개인화할 수 있는 `내 정원 표식`을 추가한 확장판입니다.

## 변경 내용

- 내 정원 무대에 개인화 표식 레이어 추가
- `내 정원 표식` 카드 추가
- 들꽃 / 조약돌 / 작은 등불 중 하나 선택 가능
- 선택한 표식이 내 나무 곁에 시각적으로 표시됨
- 선택한 표식은 localStorage에 저장되어 재방문 시 유지됨
- 표식 선택 이벤트 `garden_marker_selected` 추적 추가
- 관리자 / GA4 / Google Sheets / Apps Script / 방문자 시스템 / 서버 저장 구조는 유지

## 확인 방법

1. GitHub에 V1.25 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.25 test인지 확인
4. 내 정원으로 이동
5. `내 정원 표식` 카드가 보이는지 확인
6. 들꽃 / 조약돌 / 작은 등불 중 하나를 선택
7. 내 나무 곁에 선택한 표식이 보이는지 확인
8. 새로고침 후에도 선택한 표식이 유지되는지 확인

## 주의

- 이번 버전은 정원 개인화 확장판입니다.
- 로그인, 서버 저장, Google 계정 로그인은 추가하지 않았습니다.
- Apps Script를 새로 배포하지 않아도 됩니다.
- 기존 assets 폴더와 이미지 파일은 삭제하지 마세요.
