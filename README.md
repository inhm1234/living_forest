# 살아있는 숲

현재 테스트판: **V1.7 test**  
기준 버전: **V1.6.1 test**  
최근 업데이트: 2.5D 월드 숲 비주얼 1차  
공개 주소: https://inhm1234.github.io/living_forest/

## 프로젝트 소개

살아있는 숲은 하루에 한 번 지금의 마음을 남기면, 내 나무와 월드 숲이 조용히 자라는 웹 기반 체험판입니다.

현재 버전은 서버 저장이나 로그인 없이, 브라우저의 `localStorage`에 기록을 저장합니다. 그래서 기록은 이 기기와 이 브라우저에만 남습니다.

## 현재 포함된 기능

- 월드 숲 첫 화면
- 2.5D 월드 숲 레이어 구조 1차
- 낮 / 노을 / 밤 분위기 기초
- 월드 숲 생명체 움직임 기초
- 월드 숲 내 나무 강조 보기
- 개인 정원 이동
- 나무 이름 정하기
- 하루 1번 감정 기록
- 좋음 / 보통 / 피곤 감정에 따른 성장 반응
- 3일 / 7일 / 14일 / 21일 / 30일 성장 목표 안내
- 월드 숲 내 자리 단계 안내
- 새 / 다람쥐 방문자 애니메이션
- 오늘 남은 방문자 흔적 카드
- 최근 방문자 기록 3개 표시
- 저장 구조 `schema 3` / `local-only` 표시
- 테스트 모드 `?test=1`

## V1.7 test 핵심 변경점

V1.7 test는 기능 추가보다 월드 숲 첫 화면의 관람감을 강화하는 테스트판입니다.

- 월드 숲을 2.5D 레이어 구조로 재정리
- 뒤쪽 깊은 숲 / 중간 숲길과 빈터 / 앞쪽 풀과 안개 레이어 분리
- 프론트 기준 시간에 따라 낮 숲, 노을 숲, 밤 숲 분위기 적용
- 새, 다람쥐, 나비, 반딧불이, 빛 입자가 각각 독립 확률로 등장하는 기초 구조 추가
- “내 나무 보기” 버튼으로 월드 숲의 내 자리를 은은하게 강조
- 기존 저장키와 개인 정원 기능 유지

## V1.6.1 test에서 유지된 방문자 흔적 보완

- 방문자 판정 날짜를 한국 기준 날짜 흐름에 맞게 보정
- `오늘 남은 흔적` 카드에는 오늘 방문자 흔적만 표시
- 어제 방문자는 `숲의 방문 기록`에만 표시
- 오늘 방문자가 없으면 `오늘은 조용한 정원이에요` 문구 표시

## V1.5 test에서 유지된 저장 구조

- 저장 방식 정보를 `storageInfo`로 분리
- `dataSchemaVersion` 3 유지
- 현재 저장 방식은 `local-only` / `browser-localStorage`
- 추후 서버 저장용 자리 `serverUserId`, `serverTreeId`, `lastSyncedAt` 유지
- 기존 저장키 `livingForestV012` 유지
- 테스트 저장키 `livingForestV012_TEST` 유지
- 방문자 판정 저장 구조 유지

## V1.4 stable에서 유지된 방문자 규칙

- 오늘 기록 전에는 방문자가 자동 등장하지 않음
- 오늘 감정 기록 후 확률에 따라 방문자 판정
- 이미 오늘 기록한 사용자도, 오늘 방문자 판정이 없으면 개인 정원 입장 시 하루 1회 판정
- 오늘 방문자 판정이 이미 있으면 새로고침해도 다시 뽑지 않음

## 테스트 방법

일반 화면:

https://inhm1234.github.io/living_forest/

테스트 모드:

https://inhm1234.github.io/living_forest/?test=1

테스트 모드에서 확인할 것:

- 데이터 표시에 `V1.7 test · schema 3 · local-only`가 보이는지
- 월드 숲 첫 화면이 기존보다 더 넓고 깊은 2.5D 느낌으로 보이는지
- 낮 / 노을 / 밤 분위기 배지가 보이는지
- “내 나무 보기” 버튼을 눌렀을 때 내 자리가 강조되는지
- `방문 조건 기록 전` 버튼에서 방문자가 바로 등장하지 않는지
- `방문 조건 기록 후` 버튼에서 방문자 판정이 진행되는지
- `방문자 새` 버튼을 누르면 `깃털빛` 흔적이 보이는지
- `방문자 다람쥐` 버튼을 누르면 `도토리 흔적`이 보이는지
- `방문 기록 3개` 버튼에서 최근 방문 기록과 흔적 라벨이 보이는지
- 기존 나무 이름과 성장 기록이 이어지는지
