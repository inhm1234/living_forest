## V1.10.10 test 핵심 변경점

V1.10.10 test는 월드 숲이 조금 더 사용자가 원하는 “야구장 관중석처럼 자리가 하나씩 채워지는 공동 숲” 느낌에 가까워지도록 구조를 보정한 테스트판입니다.

핵심 방향은 다음과 같습니다.
- 중앙의 큰 무대/돔처럼 보이던 밝은 영역을 약화
- 월드 숲을 여러 줄의 숲자리 구조로 다시 정리
- 비어 있는 자리와 채워진 자리를 함께 보여 “자리들이 채워지는 감각” 강화
- 배경 숲의 존재감을 낮추고, 사용자가 키운 나무들이 월드 숲의 주인공처럼 보이게 조정
- 내 나무는 특별 전시물이 아니라 공동 숲의 한 그루처럼 섞이고, 내 나무 보기 버튼을 누를 때만 은은하게 찾을 수 있게 유지
- 개인 정원 성장, 나무 성장 이미지, 방문자 흔적, 낮/노을/밤 시간대 연동은 유지

## 이번 버전에서 확인할 것

- 중앙 화면이 전시장/무대보다 “언덕 위 숲자리”처럼 느껴지는지
- 여러 줄의 숲자리가 이전보다 더 분명하게 보이는지
- 비어 있는 자리와 채워진 자리가 함께 보여 공동체 구조가 느껴지는지
- 가까이서는 각각의 나무, 멀리서는 큰 숲처럼 느껴지는지
- 내 나무가 많은 자리 중 한 자리/한 그루로 자연스럽게 섞이는지
