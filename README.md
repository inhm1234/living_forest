# 살아있는 숲

기준 버전: V1.54.1 test  
최근 업데이트: 친구 목록에서 숲 자리 배치/이동/교체  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.55 test 핵심 변경점

V1.55 test는 `friend_links`에 저장된 친구 목록을 보고, 내 숲의 다섯 자리 중 원하는 곳에 친구를 직접 배치할 수 있게 만든 버전입니다.

## 이번에 바뀐 것

- 친구 관계 저장소 카드 안에 자리 배치 버튼 추가
- 친구 카드마다 `꽃길 / 햇살 / 연못 / 그네 / 꽃담` 버튼 표시
- 친구를 빈 자리로 배치 가능
- 이미 다른 친구가 있는 자리로 이동하려고 하면 교체 확인창 표시
- 교체된 친구는 삭제하지 않고 `friend_links` 친구 목록에 유지
- 자리에서 친구를 비우면 `friend_seats`는 `deleted` 처리하고, 친구 관계는 `friend_links`에 남김
- 기존 `friend_seats` 데이터가 있으면 `friend_links`로 1차 보정해 목록에 보이도록 처리
- 홈페이지 상단 버전, README, VERSION.txt를 V1.55 test로 맞춤

## Google Sheets 구조

- `events`: 방문/클릭 기록 저장
- `friend_seats`: 내 숲에 실제로 보이는 다섯 자리 저장
- `friend_links`: 친구 관계/친구 목록 저장
