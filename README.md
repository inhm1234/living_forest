## V1.10.4 test 핵심 변경점

V1.10.4 test는 월드 숲에 배치된 나무들이 배경 숲 위에 얹힌 느낌을 줄이고,
안개 / 전경 / 가림 레이어를 보정해 나무가 장면 속으로 더 자연스럽게 녹아들도록 만든 테스트판입니다.

## 이번 버전에서 확인할 것

- 주변 나무들이 배경 숲에 더 자연스럽게 섞여 보이는지
- 일부 나무가 풀 / 안개 / 전경에 살짝 묻히는 느낌이 나는지
- 내 나무가 너무 전시된 중심 오브젝트처럼 보이지 않는지
- 이름표와 상태 문구가 장면 감상에 덜 개입하는지
