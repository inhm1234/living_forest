## V1.10.17 test 핵심 변경점

V1.10.17 test는 사용자가 정리한 새 방향에 맞춰 화면의 역할을 분리한 테스트판입니다.

핵심 방향은 다음과 같습니다.

- 첫 화면은 내 나무를 크게 보여주는 화면이 아니라, 넓고 깊은 월드 숲 전체가 먼저 보이는 화면으로 보정
- 월드 숲의 길은 화면의 주인공이 아니라, 큰 숲 안을 흐르는 작은 길처럼 보이도록 조정
- 나무 여러 그루가 따로 놓인 느낌보다, 나무들이 모여 하나의 큰 숲을 이루는 느낌을 강화
- 내 나무 화면은 ‘정원’ 느낌보다 숲 안으로 들어간 가까운 자리처럼 보이도록 조정
- 내 나무 주변에 길, 풀, 흙, 잔디, 주변 나무가 함께 느껴지도록 CSS 레이어 보정
- 버전/캐시 쿼리를 V1.10.17 test 기준으로 정리
