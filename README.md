# 살아있는 숲

현재 테스트판: V1.28 test  
기준 버전: V1.27.1 test  
최근 업데이트: 내일의 씨앗 1차  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.28 test 핵심 변경점

V1.28 test는 V1.27.1의 `숲의 소리` 감각 확장 다음 단계로, 오늘의 기록이 다음날 다시 들어오는 이유로 이어지도록 `내일의 씨앗` 기능을 추가한 버전입니다.

## 변경 내용

- 감정 기록 후 `내일의 씨앗` 카드 활성화
- 내일의 나에게 남길 짧은 문장 저장 기능 추가
- 다음날 접속 시 어제 남긴 씨앗 문장 표시
- 같은 날에는 내일의 씨앗을 한 번만 남길 수 있도록 처리
- localStorage 기반 저장 유지
- 관리자 / GA4 / Google Sheets / Apps Script 구조는 유지

## 확인 방법

1. GitHub에 V1.28 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.28 test인지 확인
4. 내 정원에서 감정 기록 후 `내일의 씨앗` 카드가 활성화되는지 확인
5. 짧은 문장을 저장하면 카드가 완료 상태로 바뀌는지 확인
6. 테스트 모드에서 다음날 상태를 만들어 어제 남긴 씨앗이 보이는지 확인
7. 기존 숲 일기장, 공유, 초대장, 배지, 돌봄, 표식, 산책, 소리 기능이 유지되는지 확인

## 주의

- 이번 버전은 새 서버 기능이 아닙니다.
- 기록과 내일의 씨앗은 기존처럼 이 기기의 localStorage에 저장됩니다.
- Apps Script를 새로 배포하지 않아도 됩니다.
- 기존 assets 폴더와 이미지 파일은 삭제하지 마세요.
