# 살아있는 숲

현재 테스트판: V1.13 test  
기준 버전: V1.12 test  
최근 업데이트: 연속 기록 보상 강화판  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.13 test 핵심 변경점

V1.13 test는 V1.12에서 만든 “다음날 다시 왔을 때 이어지는 느낌” 다음 단계로, 계속 기록하면 작은 보상과 다음 변화가 기다린다는 느낌을 강화한 버전입니다.

## 변경 사항

- 내 정원에 `다음 숲 보상` 카드 추가
- 연속 기록 일수 기준으로 다음 보상 안내 표시
- 3일 / 7일 / 14일 / 30일 목표 보상 문구 추가
- 다음 변화까지 남은 기록 수 표시
- 진행률 바로 다음 목표까지의 거리 표시
- 감정 기록 후 “다음 보상에 가까워졌다”는 문구 보강

## 유지한 것

- 관리자 대시보드 유지
- GA4 유지
- Google Sheets / Apps Script 유지
- 방문자 시스템 유지
- 월드 숲 큰 비주얼 유지
- 기존 localStorage 저장키 유지

## 확인 방법

1. GitHub에 V1.13 파일 업로드
2. 일반 화면 접속
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.13 test인지 확인
4. 내 정원에서 기록이 있는 상태일 때 `다음 숲 보상` 카드가 보이는지 확인
5. 감정 기록 후 보상 문구와 진행률이 자연스럽게 보이는지 확인
6. 관리자 화면 숫자가 기존처럼 유지되는지 확인
