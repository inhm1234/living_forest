## V1.10.13 test 핵심 변경점

V1.10.13 test는 이전 버전에서 배경 이미지와 여러 효과가 겹치며 월드 숲이 점점 복잡하고 어색해지던 문제를 줄이기 위해 만든 재설계 테스트판입니다.

핵심 방향은 다음과 같습니다.
- 강한 배경 숲 이미지를 걷어내고, 단순한 하늘과 언덕형 무대 중심으로 재구성
- 월드 숲을 “멋진 배경 그림”이 아니라 “사용자 나무가 앞으로 채워질 빈 무대”로 표현
- 실제 사용자가 적은 초기 상태를 고려해, 나무가 듬성듬성 있어도 어색하지 않게 조정
- 빈자리는 아주 약하게, 채워진 나무는 원근감에 맞춰 보이도록 조정
- 멀리 보기 / 가까이 보기 기능은 유지하되, 새 무대에 맞춰 단순하게 재적용
- 개인 정원 성장, 나무 성장 이미지, 방문자 흔적, 낮/노을/밤 시간대 연동은 유지

## 이번 버전에서 확인할 것

- 배경이 다시 주인공처럼 보이지 않는지
- 화면이 이전보다 덜 탁하고 덜 복잡한지
- 나무 수가 적어도 “앞으로 채워질 언덕”처럼 보이는지
- 멀리 보기 / 가까이 보기에서 원근감이 조금이라도 느껴지는지
- 내 나무가 공동 숲의 한 자리처럼 자연스럽게 보이는지
