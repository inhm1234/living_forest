# 살아있는 숲 V1.72.8 test

현재 V1.72.8 test는 **전체숲과 내 정원의 시간대/색감 통일감**을 보정한 판입니다.

## 핵심 방향

- 내 정원과 전체숲이 같은 낮/노을/밤 흐름을 공유하도록 보정
- 전체숲 배경은 `assets/world/world_forest_pastel_v1_72_4.png`를 유지하되 시간대별 톤 오버레이 적용
- 전체숲의 내 나무와 주변 나무가 항상 밝은 낮 나무처럼 보이지 않도록 톤 필터 적용
- 내 정원 나무도 배경 시간대와 어긋나지 않게 낮/노을/밤 필터만 적용
- 전체숲의 한 화면 고정 무대, HUD, 하단 버튼 구조는 V1.72.5 기준 유지
- 내 정원 V1.71.6의 고정 게임 무대 구조는 유지

## 확인 포인트

1. 상단 버전이 `V1.72.8 test`로 보이는지 확인
2. 내 정원이 노을이면 전체숲도 노을 톤으로 보이는지 확인
3. 내 정원이 밤이면 전체숲도 밤 톤으로 보이는지 확인
4. 전체숲의 내 나무가 배경과 따로 노는 밝은 낮 나무처럼 보이지 않는지 확인
5. 내 정원의 나무도 노을/밤 배경에 너무 푸릇하게 튀지 않는지 확인
6. 전체숲의 `내 정원 / 내 자리 / 초대` 버튼이 유지되는지 확인
7. 긴 설명 카드/관리 카드가 전체숲 아래에 다시 나오지 않는지 확인
