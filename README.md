# 살아있는 숲

현재 테스트판: V1.30 test  
기준 버전: V1.29 test  
최근 업데이트: 주간 숲 편지 1차  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.30 test 핵심 변경점

V1.30 test는 V1.20 이후 이어진 여러 확장 기능을 다시 보강하는 버전이 아니라, 최근 기록을 모아 사용자가 며칠 동안 쌓아온 숲의 흐름을 하나의 짧은 편지처럼 보여주는 확장입니다.

## 변경 내용

- 내 정원에 `주간 숲 편지` 카드 추가
- 최근 기록 3개 이상부터 편지 자동 표시
- 최근 7개 기록 기준으로 가장 많이 남은 감정 흐름 요약
- 작은 돌봄 / 숲길 산책 / 나를 위한 작은 실천 / 내일의 씨앗이 있으면 편지 내용에 함께 반영
- 최근 기록 수, 가장 많은 마음, 작은 선택 수를 요약 칩으로 표시
- 관리자 / GA4 / Google Sheets / Apps Script / 방문자 시스템 구조는 유지

## 확인 방법

1. GitHub에 V1.30 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.30 test인지 확인
4. 기록이 3개 미만일 때는 주간 숲 편지가 준비 중으로 보이는지 확인
5. 테스트 모드에서 기록 3개 이상 상태를 만들고 주간 숲 편지가 열리는지 확인
6. 돌봄 / 산책 / 작은 실천 / 내일의 씨앗이 있으면 편지 문구와 요약 칩에 반영되는지 확인
7. 기존 숲 일기장, 공유, 초대장, 배지, 돌봄, 표식, 산책, 소리, 내일의 씨앗, 작은 실천 기능이 유지되는지 확인

- Apps Script를 새로 배포하지 않아도 됩니다.
