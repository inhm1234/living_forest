# 살아있는 숲

현재 테스트판: V1.23 test  
기준 버전: V1.22 test  
최근 업데이트: 내 숲 배지 1차  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.23 test 핵심 변경점

V1.23 test는 V1.22의 `숲 초대장` 흐름 다음 단계입니다.
이번 버전은 같은 카드나 문구를 반복 보강하지 않고, 기록과 나눔이 쌓인 흔적을 `내 숲 배지`로 보여주는 성취/수집 1차 확장판입니다.

## 변경 내용

- 내 정원 화면에 `내 숲 배지` 카드 추가
- 이름 저장, 첫 기록, 숲 일기장 생성, 3회 기록, 7회 기록, 숲 문장 나눔, 초대 링크 시작에 따라 배지 표시
- 열린 배지와 아직 잠긴 배지를 구분해 다음 행동 목표를 보여줌
- 오늘의 숲 문장을 복사/공유하면 `숲 문장 나눔` 배지가 열림
- 초대 링크에서 `내 숲 시작하기`를 누르면 `초대받은 씨앗` 배지 조건이 저장됨
- V1.20 1차 완성 흐름, V1.21 공유 기능, V1.22 초대 기능은 유지
- 로그인 / 서버 저장 / Google 계정 로그인 / 관리자 구조는 추가하지 않음

## 확인 방법

1. GitHub에 V1.23 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.23 test인지 확인
4. 내 정원에서 `내 숲 배지` 카드가 보이는지 확인
5. 이름 저장 / 감정 기록 후 첫 배지들이 열리는지 확인
6. 감정 기록을 3회 또는 7회 테스트 데이터로 만들었을 때 관련 배지가 열리는지 확인
7. 오늘의 숲 문장 복사/공유 후 `숲 문장 나눔` 배지가 열리는지 확인
8. 초대 링크로 들어와 `내 숲 시작하기`를 누르면 초대 배지 조건이 저장되는지 확인
9. 기존 감정 기록, 숲 일기장, 공유, 초대, 방문자 시스템, 관리자 숫자가 유지되는지 확인

## 주의

- 이번 버전은 기록/공유/초대의 흔적을 배지로 보여주는 프론트 확장판입니다.
- Apps Script를 새로 배포하지 않아도 됩니다.
- 기존 assets 폴더와 이미지 파일은 삭제하지 마세요.
