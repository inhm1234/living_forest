## V1.51 test 핵심 변경점

V1.51 test는 자리별 친구 초대 흐름에서 주소 복사보다 카카오톡 초대를 먼저 보여주는 버전입니다.

## 수정한 것

- Kakao JavaScript SDK 스크립트 추가
- JavaScript 키 적용
- 친구 초대 카드에 `카카오톡으로 초대하기` 버튼 추가
- 카카오톡 공유가 실패하면 초대 링크를 자동 복사하는 보조 흐름 추가
- 기존 `초대 문구 복사`를 `초대 링크 복사` 보조 기능으로 변경
- 꽃길 / 햇살 / 연못 / 그네 / 꽃담 자리별 카카오톡 초대 메시지 생성

## 확인할 것

- 카카오 Developers에서 JavaScript SDK 도메인에 `https://inhm1234.github.io`가 저장되어 있어야 합니다.
- GitHub 업로드 후 Ctrl+F5 새로고침합니다.
- 월드 숲에서 자리를 고른 뒤 `카카오톡으로 초대하기` 버튼을 눌러 확인합니다.
