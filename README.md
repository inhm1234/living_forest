# 살아있는 숲 V1.72.1 test

현재 V1.72.1 test는 **내 정원에 이어 전체숲도 고정된 게임 무대처럼 정리한 판**입니다.
전체숲 첫 화면에서 긴 설명/관리 카드는 숨기고, 사람들이 만든 나무가 모여 보이는 숲 장면을 중심에 둡니다.

## 핵심 방향

- 전체숲은 설명 페이지가 아니라 한 화면짜리 숲 무대처럼 보임
- 더미 나무는 배경 숲을 채우고, 실제 친구 나무는 라벨이 보이도록 유지
- 첫 화면에서 긴 안내 카드/관리 카드/자리 관리 UI를 숨김
- 하단 HUD는 `내 정원 / 내 자리 / 초대`만 유지
- `초대`는 팝업으로 열리고, 숲 화면을 밀어내지 않음
- 내 정원 V1.71.6의 고정 게임 무대 구조는 유지

## 확인 포인트

1. 상단 버전이 `V1.72.1 test`로 보이는지 확인
2. 전체숲에서 설명 카드가 길게 이어지지 않는지 확인
3. 스크롤 없이 숲 무대가 한 화면 안에서 보이는지 확인
4. 전체숲 하단에 `내 정원 / 내 자리 / 초대` 버튼만 보이는지 확인
5. `내 정원` 버튼으로 정원 화면 이동이 되는지 확인
6. `내 자리` 버튼으로 내 나무가 강조되는지 확인
7. `초대` 버튼이 팝업으로 열리는지 확인
8. 내 정원 화면의 `기록 / 꾸미기 / 나눔` HUD 구조가 유지되는지 확인
