## V1.10.25 test 핵심 변경점

V1.10.25 test는 V1.10.24에서 확인된 문제,
즉 **배경 숲은 좋아졌지만 내 나무가 너무 묻혀 보이는 점**을 보완하는 미세 수정판입니다.

핵심 방향은 다음과 같습니다.

- 전체 숲 화면에서 내 나무를 약간 더 잘 찾을 수 있도록 크기와 존재감을 소폭 보강
- 내 나무 이름표 가독성을 조금 개선
- 하얀 빛 / 나비 / 파티클처럼 내 나무보다 먼저 눈에 들어오던 요소를 약화
- 배경 합성 초기화 구조와 선명한 숲 배경은 그대로 유지
- 개인 정원 구조는 크게 흔들지 않고 기존 방향 유지

## 이번 버전의 목적

이 버전의 목적은 숲 배경을 다시 손대는 것이 아니라,
**숲 속에서 내 나무를 자연스럽지만 분명하게 찾을 수 있게 만드는 것**입니다.
