## V1.10.7 test 핵심 변경점

V1.10.7 test는 월드 숲이 여전히 스티커처럼 붙어 보이고, 내 떡잎과 주변 숲의 성장 차이가 약하게 느껴지던 문제를 줄이기 위해 만든 테스트판입니다.

핵심 방향은 다음과 같습니다.
- 내 떡잎은 더 작고 더 낮게 보이도록 조정
- 주변 일부 나무는 훨씬 크게 보여 숲의 거대함을 강조
- 앞 / 중간 / 뒤 레이어 차이를 더 벌려 원근감을 강화
- 작은 나무를 많이 두기보다 큰 숲의 벽과 중간 나무 무리를 만들어 깊이감 강화
- 중앙 특별 구역 느낌을 줄이고, 숲 바닥에 작은 떡잎이 붙어 있는 느낌을 강화

## 이번 버전에서 확인할 것
- 내 떡잎이 바닥에 더 가깝고 더 작게 느껴지는지
- 주변 숲이 훨씬 크고 압도적으로 느껴지는지
- 큰 나무 / 중간 나무 / 작은 나무의 차이가 확실해졌는지
- 화면이 배치도보다 숲 안 장면처럼 느껴지는지
