# 살아있는 숲 V1.73.5 focus no-move hotfix

현재 V1.73.5 focus no-move hotfix는 **전체숲에서 `내 자리` 버튼을 눌러도 내 나무 좌표가 바뀌지 않도록 최종 오버라이드한 버전**입니다.

## 수정 내용

- `내 자리` 클릭 시 `#myWorldSpot`의 left/top/bottom 좌표를 바꾸던 V1.73.4 오버라이드 무력화
- 강조 상태에서도 내 나무는 원래 자리 유지
- 나무 이미지/링/라벨만 강조
- 백서 버튼 정상 동작 유지
- 실제 시간대 자동 적용 유지

## 확인 포인트

1. 상단 버전이 `V1.73.5 hotfix · 내 자리 위치 유지`로 보이는지 확인
2. 전체숲에서 `내 자리`를 눌렀을 때 내 나무가 다른 좌표로 이동하지 않는지 확인
3. 백서 버튼이 열리는지 확인
4. 일반 URL에서 현재 시간대 숲이 자동 적용되는지 확인
