# 살아있는 숲

현재 테스트판: V1.22 test  
기준 버전: V1.21 test  
최근 업데이트: 숲 초대장 1차  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.22 test 핵심 변경점

V1.22 test는 V1.21에서 추가한 `오늘의 숲 문장 공유` 다음 단계입니다.  
이번 버전은 같은 화면을 반복 보강하지 않고, 공유된 숲 문장을 보고 들어온 사람이 자연스럽게 `내 숲 시작하기`로 이어지도록 만든 초대/유입 1차 확장판입니다.

## 변경 내용

- 공유 문장 링크를 `?invite=forest_sentence` 초대 링크로 변경
- 초대 링크로 들어온 방문자에게 `숲 초대장` 카드 표시
- `숲 초대장` 카드에서 살아있는 숲의 핵심 흐름을 짧게 안내
- 초대 방문자가 바로 `내 숲 시작하기`를 누를 수 있는 CTA 추가
- 초대 CTA 클릭은 기존 관리자 집계 흐름의 `go_garden_click` 이벤트로 기록
- V1.20 1차 완성 흐름과 V1.21 오늘의 숲 문장 공유 기능은 유지
- 로그인 / 서버 저장 / Google 계정 로그인 / 관리자 구조는 추가하지 않음

## 확인 방법

1. GitHub에 V1.22 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.22 test인지 확인
4. 감정 기록 후 `오늘의 숲 나누기` 카드에서 공유/복사 문장이 생성되는지 확인
5. 복사된 링크가 `?invite=forest_sentence`를 포함하는지 확인
6. 초대 링크로 접속했을 때 `숲 초대장` 카드가 보이는지 확인
7. `내 숲 시작하기` 버튼으로 내 정원 이동이 되는지 확인
8. 기존 감정 기록, 숲 일기장, 방문자 시스템, 관리자 숫자가 유지되는지 확인

## 주의

- 이번 버전은 공유 이후의 초대/유입 흐름을 붙인 프론트 확장판입니다.
- Apps Script를 새로 배포하지 않아도 됩니다.
- 기존 assets 폴더와 이미지 파일은 삭제하지 마세요.
