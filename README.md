# 살아있는 숲

현재 테스트판: V1.42 test  
기준 버전: V1.41.1 test  
최근 업데이트: 귀여운 숲 비주얼 개편 1차  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.42 test 핵심 변경점

V1.42 test는 리얼하고 어두운 숲 분위기에서 한 걸음 벗어나, 여자아이들도 더 편하게 들어와 놀고 꾸미고 친구 숲을 떠올릴 수 있도록 파스텔 / 웹툰풍 톤으로 전환하는 비주얼 개편 1차입니다.

이번 버전은 전체 기능 구조를 크게 뜯기보다, 기존 월드 숲 / 내 정원 / 하단 패널 흐름을 유지한 채 첫인상과 UI 톤을 먼저 바꾸는 데 집중했습니다.

## 수정한 것

- 일반 화면과 내 정원의 버전 표기를 V1.42 test로 갱신
- world / garden 소개 문구를 `혼자 기록하는 숲`보다 `함께 채워가는 친구 숲` 방향으로 조정
- 전체 배경을 어두운 숲 그라데이션에서 크림 / 핑크 / 라벤더 / 민트 중심 파스텔 톤으로 변경
- visual-card / glass-panel / mood-card / forest-head / garden-hub 등 주요 카드의 색감과 그림자 분위기를 밝고 말랑하게 변경
- 메인 버튼을 핑크 + 라벤더 중심의 둥근 버튼 톤으로 변경
- 보조 버튼 / 탭 버튼 / 입력창 / 메타칩을 밝은 크림/화이트 바탕의 귀여운 UI 톤으로 변경
- 월드 숲 무대와 내 정원 무대의 오버레이/패널 색을 밝게 조정해 귀여운 숲 비주얼로 넘어가기 위한 기반을 마련
- README / VERSION / 업로드 가이드 / 체크리스트를 V1.42 기준으로 정리
- 다음 단계에서 교체할 v2 배경/나무 자산 목록 가이드를 추가

## 이번 버전에서 유지한 것

- 월드 숲 / 내 정원 / 감정 기록 구조
- 하단 패널 기본 흐름 (기록 / 활동 / 꾸미기 / 편지 / 보관함)
- 숲 일기장
- 주간 숲 편지
- 오늘의 작은 돌봄
- 정원 표식
- 숲길 산책
- 숲의 소리
- 내일의 씨앗
- 나를 위한 작은 실천
- 내 숲 보관 파일 받기 / 불러오기 / 복원
- 숲 달력 / 내 숲 기억 찾기
- 배지 / 공유 / 초대장 / 방문자 시스템
- 관리자 / GA4 / Google Sheets / Apps Script 구조

## 아직 이번 버전에서 하지 않은 것

- 실제 친구 로그인 / 실시간 공동 숲
- 완전한 꾸미기 아이템 수집 시스템
- 나무 성장 이미지 전체 v2 교체
- 월드 / 내 정원 배경 이미지의 완전한 웹툰풍 교체

## 주의

- 이번 버전은 `비주얼 개편 1차`이므로, 구조와 기능은 유지하고 색감/카드/버튼 중심으로 먼저 바꾼 버전입니다.
- 복원 기능은 선택한 보관 파일로 현재 이 기기의 숲 기록을 덮어쓸 수 있습니다.
- 복원 전에는 현재 숲을 먼저 `보관 파일 받기`로 내려받는 것을 추천합니다.
- `assets` 폴더는 포함되어 있으며, 다음 단계에서 `v2` 배경/나무 이미지로 일부 교체할 수 있도록 준비되어 있습니다.
