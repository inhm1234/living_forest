# 살아있는 숲

현재 테스트판: V1.24 test  
기준 버전: V1.23 test  
최근 업데이트: 오늘의 작은 돌봄 1차  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.24 test 핵심 변경점

V1.24 test는 V1.23의 `내 숲 배지` 다음 단계입니다. 기존 흐름을 다시 보강하지 않고, 감정 기록을 마친 뒤 내 나무에게 하루 한 번 작은 돌봄 행동을 남기는 새 상호작용 축을 추가했습니다.

## 변경 내용

- 내 정원 화면에 `오늘의 작은 돌봄` 카드 추가
- 감정 기록 후 `물 주기 / 빛 쬐기 / 쉬게 하기` 중 하나 선택 가능
- 하루에 한 번만 돌봄 선택 가능
- 선택한 돌봄이 localStorage의 `careHistory`에 저장됨
- 선택 후 오늘의 돌봄 문구가 카드와 메시지에 표시됨
- `tree_care_selected` 이벤트로 선택 행동 추적
- 관리자 / GA4 / Google Sheets / Apps Script / 방문자 시스템 구조는 유지

## 확인 방법

1. GitHub에 V1.24 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.24 test인지 확인
4. 내 정원에서 감정 기록을 완료
5. `오늘의 작은 돌봄` 카드에서 물 주기 / 빛 쬐기 / 쉬게 하기 중 하나 선택
6. 선택 후 카드 문구가 바뀌고, 같은 날에는 다시 선택할 수 없는지 확인
7. 숲 일기장, 공유, 초대장, 배지, 방문자 기록이 기존처럼 유지되는지 확인

## 주의

- 이번 버전은 프론트 화면 확장판입니다.
- Apps Script를 새로 배포하지 않아도 됩니다.
- 기존 assets 폴더와 이미지 파일은 삭제하지 마세요.
