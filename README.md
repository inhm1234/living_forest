## V1.10.8 test 핵심 변경점

V1.10.8 test는 월드 숲을 “내 나무를 크게 보여주는 무대”가 아니라, 야구장 관중석처럼 여러 자리와 여러 나무가 모여 하나의 큰 숲을 이루는 방향으로 구조를 잡은 테스트판입니다.

핵심 방향은 다음과 같습니다.
- 개인 정원에서 키운 내 나무가 월드 숲의 한 자리를 채우는 느낌 강화
- 가까이서는 각각의 나무, 멀리서는 거대한 공동 숲처럼 보이도록 공동 나무 줄 구조 추가
- 기존 큰 배경 나무는 유지하되, 그 사이에 여러 줄의 커뮤니티 나무를 배치
- 내 나무는 특별한 카드가 아니라 많은 나무 중 한 그루처럼 보이게 조정
- “내 나무 보기”를 눌렀을 때만 내 나무 이름과 빛이 은은하게 강조되도록 조정
- 기존 localStorage 저장, schema 3, 방문자 흔적/방문 기록, 시간대 연동은 유지

## 이번 버전에서 확인할 것

- 월드 숲이 빈 무대가 아니라 여러 나무가 모인 공동체 숲처럼 느껴지는지
- 내 나무가 혼자 튀는 전시물이 아니라 많은 나무 중 한 그루처럼 보이는지
- 내 나무 보기 버튼을 눌렀을 때만 “저게 내 나무구나” 하고 찾을 수 있는지
- 개인 정원에서 내 나무를 키우는 경험이 여전히 중심으로 느껴지는지
- 성장 단계별 이미지가 월드와 정원에서 모두 자연스럽게 보이는지
