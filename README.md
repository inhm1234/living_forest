기준 버전: V1.25.2 test

공개 주소: https://inhm1234.github.io/living_forest/

## V1.25.3 test 핵심 변경점

V1.25.3 test는 V1.25.2에서 나무 가까이에 배치한 `오늘의 작은 돌봄` 빠른 선택 패널이 정원 화면 중앙과 하단 메시지를 가려, 사용자가 나무 변화를 바로 보기 어려웠던 문제를 수정한 버전입니다.

## 변경 내용

- 정원 무대 안 돌봄 패널을 중앙 하단 오버레이에서 우측 상단의 작은 컨트롤 박스로 이동
- 나무와 하단 성장 메시지를 가리지 않도록 패널 크기 축소
- 버튼은 `물 / 빛 / 쉼` 3개를 유지하되 더 작고 가볍게 표시
- 선택 후 결과 문구도 작은 상태 문구로만 표시
- 모바일에서도 정원 중앙을 가리지 않도록 상단 고정형으로 보정
- 기존 아래쪽 `오늘의 작은 돌봄` 카드와 기록 저장 구조는 유지

## 확인 방법

1. GitHub에 V1.25.3 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.25.3 test인지 확인
4. 내 정원으로 이동 후 감정 기록
5. 정원 무대 우측 상단에 작은 돌봄 컨트롤이 보이는지 확인
6. 물 / 빛 / 쉼 버튼을 눌러도 나무와 하단 성장 문구가 가려지지 않는지 확인
7. 아래쪽 돌봄 카드에도 같은 결과가 반영되는지 확인
