## V1.51.1 test 핵심 변경점

V1.51.1 test는 카카오톡 초대 메시지 안에 직접 초대 링크가 더 확실히 들어가도록 고치고, 처음 온 친구가 닉네임 / 나무 이름 / 오늘의 마음을 정한 뒤 초대받은 자리에 들어가는 흐름을 추가한 버전입니다.

## 수정한 것

- 카카오톡 공유 방식을 이미지 카드 중심에서 직접 링크가 포함된 텍스트 초대장 중심으로 변경
- 초대 링크에 join=1 값을 추가하여 초대받은 시작 화면을 더 명확하게 구분
- 친구 초대 카드에 직접 초대 링크 표시 영역 추가
- 초대받은 친구 카드에 시작 순서 안내 추가
- 친구가 처음 들어오면 닉네임, 나무 이름, 오늘의 마음을 정하고 저장 가능
- 친구의 첫 마음 기록이 localStorage에도 남도록 보정
