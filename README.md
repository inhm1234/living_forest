## V1.46.2 test 핵심 변경점

V1.46.2 test는 V1.46.1 화면에서 보였던 두 가지 문제를 보정한 버전입니다.

1. 내 정원 배경이 월드 숲처럼 보이던 문제를 줄이고, 더 개인 정원형 배경으로 교체했습니다.
2. 장식 이미지에 흰 사각형 배경이 보일 수 있는 문제를 줄이기 위해 새 파일명과 투명 처리된 장식 파일을 사용합니다.
