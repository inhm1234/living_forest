## V1.48 test 핵심 변경점

V1.48 test는 친구가 아직 없는 초기 숲을 보정한 다음 단계로, 사용자가 실제로 친구에게 숲을 보내고 싶어지는 초대 흐름을 추가한 버전입니다.

이번 버전에서는 월드 숲 화면에 `친구 초대 준비` 카드를 추가하고, 내 나무 이름 / 기록 일수 / 오늘의 분위기를 담은 초대 문구를 미리 보거나 복사할 수 있게 했습니다.

## 수정한 것

- 일반 화면과 내 정원의 버전 표기를 V1.48 test로 갱신
- 월드 숲 화면에 친구 초대 카드 추가
- 초대 문구 미리보기 기능 추가
- 초대 문구 복사 기능 추가

- 실제 친구 서버 연결 없이도 친구가 자기 숲을 시작할 수 있는 공유 흐름 구성
