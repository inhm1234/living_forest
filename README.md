# 살아있는 숲

현재 테스트판: V1.29 test  
기준 버전: V1.28 test  
최근 업데이트: 나를 위한 작은 실천 1차  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.29 test 핵심 변경점

V1.29 test는 V1.28의 `내일의 씨앗` 다음 확장으로, 감정 기록이 숲 안에만 남는 것이 아니라 실제 사용자의 몸과 마음으로 돌아오는 작은 행동을 하나 남기도록 만든 버전입니다.

## 변경 내용

- 내 정원 화면에 `나를 위한 작은 실천` 카드 추가
- 감정 기록 후 숨 고르기 / 물 한 모금 / 어깨 풀기 중 하나 선택 가능
- 하루에 한 번만 선택되도록 제한
- 선택한 실천은 오늘 기록으로 localStorage에 저장
- 새로고침 후에도 오늘의 작은 실천 결과 유지
- 관리자 / GA4 / Google Sheets / Apps Script 구조는 유지

## 확인 방법

1. GitHub에 V1.29 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.29 test인지 확인
4. 내 정원에서 감정 기록 후 `나를 위한 작은 실천` 카드가 활성화되는지 확인
5. 숨 고르기 / 물 한 모금 / 어깨 풀기 중 하나를 선택했을 때 결과 문구가 남는지 확인
6. 새로고침 후에도 오늘 선택한 실천이 유지되는지 확인
7. 기존 숲 일기장, 공유, 초대장, 배지, 돌봄, 표식, 산책, 소리, 내일의 씨앗 기능이 유지되는지 확인
