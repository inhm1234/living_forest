## V1.48.1 test 핵심 변경점

V1.48.1 test는 월드 숲 안의 빈자리가 5개처럼 보이는 구조에 맞춰, 숲 친구 배지도 5개로 늘리고 위치를 다시 정렬한 보정판입니다.

## 수정한 것

- 버전 표기를 V1.48.1 test로 갱신
- 월드 숲 안의 숲 친구 배지를 3개에서 5개로 변경
- 리본새싹 / 민트새소리 / 별꽃잎 / 달빛가랜드 / 소풍바람 배지 추가 및 정렬
- 배지가 내 나무 주변에 몰리지 않도록 분산 배치
- 요약 카드의 친구 자리 칩도 5개로 확장
- 친구 숲 미리보기 카드도 5개 기준으로 보이도록 조정
