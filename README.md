# 살아있는 숲 V1.73.7 neighbor position-size rebalance hotfix

현재 V1.73.7 neighbor position-size rebalance hotfix는 **전체숲 주변 나무의 위치와 크기를 다시 조정한 버전**입니다.

## 이번 수정

- 주변 나무 위치를 더 자연스럽게 재배치
- 주변 나무를 기준 배경나무에 더 가깝게 크게 확대
- 뒤쪽 나무도 더 잘 보이도록 확대
- 내 나무 위치/크기는 기존 마음에 든 상태 최대한 유지
- 내 자리 위치 고정 유지
- 백서 버튼 / 실제 시간대 자동 적용 유지
