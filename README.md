## V1.46 test 핵심 변경점

V1.46 test는 월드 숲 배경과 메인 나무를 귀여운 톤으로 바꾼 뒤, 정원 안의 작은 친구들과 꾸미기 요소도 같은 그림체로 맞추는 단계입니다.

이번 버전에서는 새 / 다람쥐 방문자 이미지를 v2로 교체하고, 정원 표식 영역을 리본 / 가랜드 / 꽃 랜턴 / 소풍 세트 장식으로 개편했습니다.

## 수정한 것

- 일반 화면과 내 정원의 버전 표기를 V1.46 test로 갱신
- 정원 무대 방문자 이미지를 `bird-silhouette-v2.png`, `squirrel-silhouette-v2.png`로 교체
- 월드 숲 작은 생명 표시에도 v2 새/다람쥐 이미지를 사용하도록 연결
- 정원 표식 카드를 귀여운 장식 카드로 전환
- 기존 들꽃 / 조약돌 / 작은 등불 선택을 리본 / 가랜드 / 꽃 랜턴 / 소풍 세트 선택으로 변경
- 장식 선택 시 정원 무대에 해당 장식 이미지가 실제로 보이도록 연결
- 장식 선택 버튼을 이미지 썸네일 카드 형태로 개편
- README / VERSION / 체크리스트 / 업로드 가이드를 V1.46 기준으로 정리
