# 살아있는 숲

현재 테스트판: V1.16 test  
기준 버전: V1.15 test  
최근 업데이트: 숲 일기장 1차  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.16 test 핵심 변경점

V1.16 test는 V1.15에서 만든 `월드 숲 자리감` 다음 단계로, 사용자가 감정을 고른 뒤 그 기록이 하루짜리 클릭으로 끝나지 않고 `숲 일기장`에 쌓이도록 만든 버전입니다.

이번 방향은 월드 숲 비주얼을 계속 보정하는 작업이 아니라, 사용자가 “내 마음이 숲에 남았다”는 느낌을 받을 수 있도록 기록의 의미를 강화하는 작업입니다.

## 변경 내용

- 내 정원 화면에 `숲 일기장` 카드 추가
- 감정 기록 후 `오늘의 숲 문장` 표시
- 최근 3개의 숲 기록 표시
- 각 기록에 날짜 / 감정 / 숲 기록 제목 / 짧은 문장 표시
- 최근 감정 흐름 요약 표시
- 기존 localStorage 기록을 그대로 활용
- 기존 관리자 / GA4 / Google Sheets / Apps Script 구조는 유지
- 로그인 / Google 계정 / 서버 저장 기능은 추가하지 않음

## 확인 방법

1. GitHub에 V1.16 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.16 test인지 확인
4. 내 나무 키우러 가기 또는 내 정원 둘러보기로 이동
5. 감정 기록 후 `숲 일기장` 카드가 보이는지 확인
6. 오늘의 숲 문장과 최근 숲 기록 3개가 자연스럽게 보이는지 확인
7. 기존 내 정원 이동, 감정 기록, 방문자 기록, 관리자 숫자 표시가 유지되는지 확인

## 주의

- 이번 버전은 프론트 화면 개선판입니다.
- Apps Script를 새로 배포하지 않아도 됩니다.
- 기존 assets 폴더와 이미지 파일은 삭제하지 마세요.
