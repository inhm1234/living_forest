# 살아있는 숲

현재 검증판: V1.56 test  
기준 안정판: V1.55 stable  
최근 업데이트: 친구 배치 UX 정리  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.56 test 핵심 정리

V1.56 test는 V1.55 stable에서 이미 성공한 친구 배치/이동/교체 기능을 더 쉽게 쓰도록 정리한 UX 패치입니다. 저장 구조와 Apps Script 동작은 크게 바꾸지 않고, 화면에서 현재 자리/빈 자리/교체 자리를 더 명확하게 보여주는 것이 목표입니다.

## 이번에 바뀐 것

- 친구 관계 저장소 상단에 `자리 현황` 안내 추가
- 다섯 자리별로 `비어 있음` / `누가 있음` 상태 표시
- 친구 카드의 자리 버튼 문구를 `현재 자리`, `빈 자리`, `교체`로 구분
- 이미 배치된 친구 카드를 더 잘 보이게 정리
- 친구 목록은 배치된 친구가 먼저 보이도록 정렬
- 교체 확인창 문구를 더 쉽게 설명하도록 개선
- 홈페이지 상단 버전, README, VERSION.txt를 `V1.56 test`로 통일

## Google Sheets 구조

- `events`: 방문/클릭 기록 저장
- `friend_seats`: 내 숲에 실제로 보이는 다섯 자리 저장
- `friend_links`: 친구 관계/친구 목록 저장

## Apps Script 작업

이번 V1.56 test는 화면 UX 중심 패치입니다.  
`APPS_SCRIPT_CODE.txt` 전체 교체와 Apps Script 새 배포는 필요하지 않습니다.

단, 만약 이전에 V1.55 stable Apps Script를 배포하지 않았다면 먼저 V1.55 stable Apps Script가 적용되어 있어야 합니다.

## V1.56 test 확인 체크리스트

1. 홈페이지 상단이 `V1.56 test`인지 확인
2. `README.md`가 `V1.56 test`인지 확인
3. `VERSION.txt`가 `V1.56 test`인지 확인
4. 친구 관계 저장소 상단에 자리 현황이 보이는지 확인
5. 각 자리가 `비어 있음` 또는 `친구 이름 있음`으로 보이는지 확인
6. 친구 카드 버튼에 `현재 자리`, `빈 자리`, `교체` 표시가 나오는지 확인
7. 빈 자리 배치가 되는지 확인
8. 교체 버튼을 누르면 확인창 문구가 이해하기 쉽게 나오는지 확인
9. 교체 후 친구 목록이 유지되는지 확인
