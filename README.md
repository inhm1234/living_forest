# 살아있는 숲

현재 테스트판: **V1.8.2 test**  
기준 버전: **V1.8.1 test**  
최근 업데이트: 낮 숲 밝기 확실 강화  
공개 주소: https://inhm1234.github.io/living_forest/

## 프로젝트 소개

살아있는 숲은 하루에 한 번 지금의 마음을 남기면, 내 나무와 월드 숲이 조용히 자라는 웹 기반 체험판입니다.

현재 버전은 서버 저장이나 로그인 없이, 브라우저의 `localStorage`에 기록을 저장합니다. 그래서 기록은 이 기기와 이 브라우저에만 남습니다.

## V1.8.2 test 핵심 변경점

V1.8.2 test는 V1.8.1 test에서 사용자가 지적한 “낮이 여전히 어둡고 밤처럼 보인다”는 문제를 보완한 테스트판입니다.

- 낮 숲 전체 밝기를 크게 올림
- 햇빛, 하늘빛, 빛줄기, 밝은 초록 톤 강화
- 중앙 숲길 / 빈터 / 내 자리 시인성 강화
- 어두운 전경과 안개를 약화해 낮답게 보이도록 조정
- 노을은 따뜻한 빛과 긴 그림자 느낌 유지
- 밤은 가장 어둡고 몽환적인 축으로 유지

## 유지되는 기능

- 개인 정원 이동
- 나무 이름 정하기
- 하루 1번 감정 기록
- 좋음 / 보통 / 피곤 감정에 따른 성장 반응
- 월드 숲 내 자리 반영
- 새 / 다람쥐 방문자 애니메이션
- 오늘 남은 방문자 흔적 카드
- 최근 방문자 기록 3개 표시
- 저장 구조 `schema 3` / `local-only`
- 테스트 모드 `?test=1`

## 테스트 방법

일반 화면:

https://inhm1234.github.io/living_forest/

테스트 모드:

https://inhm1234.github.io/living_forest/?test=1

시간대 강제 확인:

- 낮: `?test=1&worldTime=day`
- 노을: `?test=1&worldTime=sunset`
- 밤: `?test=1&worldTime=night`

테스트 모드에서 확인할 것:

- 데이터 표시에 `V1.8.2 test · schema 3 · local-only`가 보이는지
- 낮 숲이 확실히 밝고 낮처럼 느껴지는지
- 숲길 / 빈터 / 내 자리 시인성이 더 좋아졌는지
- 노을 / 밤이 낮과 확실히 구분되는지
- 기존 나무 이름과 성장 기록이 이어지는지
