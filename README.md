# 살아있는 숲

현재 검증판: **V1.59 test**

## V1.59 test 핵심 정리

V1.59 test는 **첫 방문 UX/UI 2차 정리판**입니다.  
V1.58에서 추가한 온보딩을 실제 첫 화면 기준으로 다시 다듬어, 처음 온 사람이 안내창만 보는 것이 아니라 뒤의 숲 분위기도 느끼면서 바로 첫 행동으로 들어가도록 보완했습니다.

### 이번 버전에서 바뀐 점

- 온보딩 배경 흐림을 줄여 **뒤의 숲 분위기**가 더 보이게 조정
- 온보딩의 **내 숲 시작하기** 버튼을 더 강하게 강조
- 보조 버튼은 작고 덜 튀게 바꿔서 선택지가 많아 보이지 않게 정리
- “오늘의 기록 시작하기” 클릭 후 **나무 이름 입력 위치로 자연스럽게 이동**하도록 보완
- 카드/입력칸/안내 문구의 가독성과 대비 보정
- 기존 카카오 초대, 친구 자리, 방문자, 보관함 기능은 유지

## 출시 방향 메모

이 프로젝트의 현재 목표는 지인 홍보형 테스트가 아니라,  
**블로그 소개글 / 선택적 유튜브 영상 / 검색 유입**으로 처음 온 사람이 혼자 이해하고 첫 행동을 할 수 있는 MVP입니다.

그래서 이번 버전의 우선순위는 기능 추가보다 다음 기준에 있습니다.

1. 처음 온 사람이 10초 안에 무엇을 해야 하는지 알기
2. 글씨가 잘 보이고 버튼이 명확하기
3. 타겟층에 맞는 귀엽고 밝은 UX/UI로 보이기
4. 첫 행동인 “오늘의 기록”까지 자연스럽게 이동하기

## 업로드 후 확인 체크리스트

1. 상단 버전이 `V1.59 test`로 보이는지 확인
2. 첫 방문 상태에서 30초 안내 모달이 뜨는지 확인
3. 온보딩 뒤의 숲 배경이 너무 뭉개지지 않고 분위기가 보이는지 확인
4. “내 숲 시작하기” 버튼이 가장 눈에 띄는지 확인
5. “오늘의 기록 시작하기” 버튼을 누르면 내 정원 화면으로 이동하고 나무 이름 입력 위치로 내려가는지 확인
6. 정원 화면에서 나무 이름과 마음 선택 순서가 이해되는지 확인
7. 카카오 초대/친구 자리 기능이 기존처럼 유지되는지 확인
