# 살아있는 숲

현재 테스트판: V1.12 test  
기준 버전: V1.11 test  
최근 업데이트: 다음날 재방문 경험 강화판  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.12 test 핵심 변경점

V1.12 test는 V1.11에서 만든 “내일 다시 오면 이어진다”는 기대감을 실제 재방문 화면에서 느낄 수 있도록 정리한 버전입니다.

## 변경 내용

- 어제 또는 이전 기록이 있는 사용자가 오늘 아직 기록하지 않았을 때 `이어지는 숲` 카드 표시
- 어제 남긴 감정과 오늘의 기록이 이어진다는 문구 추가
- 연속 기록 일수 느낌을 보여주는 작은 배지 추가
- 재방문자에게는 감정 선택 안내 문구를 “어제의 마음 위에 오늘의 마음을 더하기” 방향으로 변경
- 감정 기록 후에는 이전 기록 위에 오늘의 마음이 더해졌다는 문구 보강
- 관리자 / GA4 / Google Sheets / Apps Script 구조는 유지

## 확인 방법

1. GitHub에 V1.12 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.12 test인지 확인
4. 어제 기록이 있는 상태에서 오늘 접속했을 때 `이어지는 숲` 카드가 보이는지 확인
5. 감정 기록 후 오늘의 변화 / 내일 이어질 성장 카드가 기존처럼 보이는지 확인
6. 관리자 화면 숫자가 기존처럼 계속 찍히는지 확인

## 주의
