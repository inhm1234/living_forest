## V1.10.16 test 핵심 변경점

V1.10.16 test는 중앙에 사람이 걸을 수 있을 것 같은 숲길을 더 분명하게 보이도록 만들고, 나무를 그 길 양옆에 심겨 있는 구조로 다시 배치한 테스트판입니다.

핵심 방향은 다음과 같습니다.
- 중앙 흙길을 더 분명한 형태와 색으로 보이게 조정
- 길은 아래쪽에서 넓고 위쪽으로 갈수록 좁아지는 사람의 산책길 느낌으로 보정
- 나무는 길 한가운데가 아니라 양옆의 흙/잔디 자리에서 자라도록 재배치
- 길/잔디/풀/흙의 경계를 더 분명하게 하여 바닥감 강화
- 밝은 안개 오버레이를 줄여 바닥 정보가 묻히지 않게 조정
- 가까운 나무는 더 크고 선명하게, 먼 나무는 더 작고 흐리게 하여 원근감 보정
