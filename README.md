# 살아있는 숲

현재 테스트판: V1.31 test  
기준 버전: V1.30 test  
최근 업데이트: 내 숲 보관함 1차  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.31 test 핵심 변경점

V1.31 test는 V1.30의 주간 숲 편지 이후, 반복 보강이 아니라 새로운 확장 축으로 `내 숲 보관함`을 추가한 버전입니다.

이번 버전의 목적은 로그인이나 서버 저장을 바로 넣는 것이 아니라, 현재 이 기기에 저장된 내 숲 기록을 사용자가 직접 파일로 보관할 수 있게 만드는 것입니다.

## 추가 내용

- 내 정원 화면에 `내 숲 보관함` 카드 추가
- 현재 localStorage에 저장된 나무/감정/숲 일기장/작은 선택 기록을 JSON 파일로 내보내기
- 내 숲 보관 요약 복사 기능 추가
- 서버 전송 없이 사용자가 직접 보관하는 방식으로 구성
- 관리자 / GA4 / Google Sheets / Apps Script 구조는 유지

## 확인 방법

1. GitHub에 V1.31 파일 업로드
2. 일반 화면 접속
3. 화면 상단 버전이 V1.31 test인지 확인
4. 내 정원에서 `내 숲 보관함` 카드 확인
5. `보관 파일 받기` 버튼으로 JSON 파일 다운로드 확인
6. `요약 복사` 버튼으로 보관 요약 복사 확인

## 주의

- 이번 버전은 백업/보관 확장의 1차입니다.
- 가져오기/복구 기능은 아직 포함하지 않았습니다.
- Apps Script를 새로 배포하지 않아도 됩니다.
- 기존 assets 폴더와 이미지 파일은 삭제하지 마세요.
