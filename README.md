# living_forest

현재 테스트판: **V1.3 test**  
최근 업데이트: **방문자 기록 남기기 1차**  
공개 주소: https://inhm1234.github.io/living_forest/

## 프로젝트 소개

**살아있는 숲**은 하루에 한 번 지금의 마음을 남기면, 내 나무와 월드 숲이 조용히 자라는 작은 웹 체험판입니다.

좋음, 보통, 피곤 중 하나를 고르면 어떤 선택도 성장으로 남고, 나무는 죽지 않습니다.  
현재 기록은 서버가 아니라 **사용자의 기기 localStorage**에 저장됩니다.

## 현재 포함된 기능

- 월드 숲 첫 화면
- 개인 정원 이동 흐름
- 나무 이름 정하기
- 하루 1번 감정 기록
- 나무 성장 단계 안내
- 다음 성장 목표 안내
- 월드 숲 내 자리 단계 안내
- 새/다람쥐 방문자 애니메이션
- 최근 방문자 기록 3개 표시
- 테스트 모드: `?test=1`
