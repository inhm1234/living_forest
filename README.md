## V1.10.11 test 핵심 변경점

V1.10.11 test는 월드 숲의 관중석형 구조는 유지하되, 화면이 너무 게임판/전시장처럼 보이던 문제를 줄이기 위한 테스트판입니다.

핵심 방향은 다음과 같습니다.
- 중앙의 큰 밝은 타원형 무대 느낌을 약화
- 여러 줄의 숲자리 구조는 유지하되, 더 자연스럽게 흐트러져 보이도록 보정
- 비어 있는 자리를 동그란 마커보다 작은 흙자리/빈 터처럼 보이게 조정
- 나무들이 조금 더 겹치고 일부는 배경에 스며들게 하여, 멀리서 보면 하나의 숲 덩어리처럼 느껴지도록 보정
- 내 나무는 특별 전시물이 아니라 공동 숲의 한 그루처럼 보이게 하고, 내 나무 보기 버튼을 눌렀을 때만 더 분명하게 찾을 수 있게 유지
- 개인 정원 성장, 나무 성장 이미지, 방문자 흔적, 낮/노을/밤 시간대 연동은 유지

## 이번 버전에서 확인할 것

- 중앙 화면이 돔/무대보다 자연스러운 언덕처럼 느껴지는지
- 비어 있는 자리들이 좌석 마커보다 작은 흙자리처럼 보이는지
- 줄 구조는 느껴지지만, 너무 규칙적인 게임판처럼 보이지 않는지
- 나무들이 따로 놓인 느낌보다 하나의 숲 덩어리처럼 느껴지는지
- 내 나무가 많은 자리 중 한 자리/한 그루로 자연스럽게 섞이는지
