## V1.47 test 핵심 변경점

V1.47 test는 비주얼 개편 이후 다음 단계로, 예쁜 숲을 넘어 `친구와 함께 키우는 숲` 감각이 더 잘 느껴지도록 월드 숲 카드와 문구를 보강하는 버전입니다.

이번 버전에서는 실제 로그인이나 서버 기능을 넣지 않고도, 내 자리 주변의 빈자리가 `친구가 들어올 자리`처럼 보이도록 안내를 강화하고, 월드 숲 화면에 친구 숲 미리보기 카드를 추가했습니다.

## 수정한 것

- 일반 화면과 내 정원의 버전 표기를 V1.47 test로 갱신
- 월드 숲 자리 안내 문구를 `친구가 들어올 자리` 중심으로 조정
- `친구 숲의 움직임` 카드 추가
- 월드 숲에서 내 기록 수에 따라 달라지는 친구 자리 미리보기 3개 표시
- 친구 숲 / 함께 키우는 감각을 강조하는 문구로 world summary와 community hint를 보강
- README / VERSION / 체크리스트 / 업로드 가이드를 V1.47 기준으로 정리
