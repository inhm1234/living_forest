# 살아있는 숲

현재 테스트판: V1.26 test  
기준 버전: V1.25.3 test  
최근 업데이트: 숲길 산책 1차  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.26 test 핵심 변경점

V1.26 test는 V1.20 1차 완성형 이후의 추가 확장 흐름입니다. 기존 숲 일기장, 공유, 초대장, 배지, 작은 돌봄, 정원 표식 흐름을 반복 보강하지 않고, 감정 기록 후 내 숲 안에서 오늘 걸을 길을 하나 고르는 `숲길 산책` 축을 추가했습니다.

## 변경 내용

- 내 정원 화면에 `오늘의 숲길 산책` 카드 추가
- 감정 기록 후 이끼길 / 햇살길 / 고요한 길 중 하나 선택 가능
- 하루에 한 번만 산책 선택 가능
- 선택한 숲길 산책 기록을 localStorage에 저장
- 기록 전에는 최근 산책 흔적 안내, 기록 후에는 오늘 산책 결과 표시
- 숲길 산책 선택 이벤트 `forest_trail_selected` 추적 추가
- 관리자 / GA4 / Google Sheets / Apps Script / 방문자 시스템 / 로그인 / 서버 저장 구조는 변경하지 않음

## 확인 방법

1. GitHub에 V1.26 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.26 test인지 확인
4. 내 정원에서 감정 기록 후 `오늘의 숲길 산책` 카드 확인
5. 이끼길 / 햇살길 / 고요한 길 중 하나 선택
6. 선택 후 결과 문구가 표시되고 같은 날 다시 선택되지 않는지 확인
7. 기존 숲 일기장, 공유, 초대장, 배지, 돌봄, 정원 표식, 방문자 기능이 유지되는지 확인

## 주의

- 이번 버전은 추가 확장판입니다.
- 기존 기능을 반복 보강하는 작업이 아니라, 기록 후 선택할 수 있는 새로운 산책 축을 추가했습니다.
- Apps Script를 새로 배포하지 않아도 됩니다.
- 기존 assets 폴더와 이미지 파일은 삭제하지 마세요.
