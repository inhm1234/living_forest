## V1.10.28 test 핵심 변경점

운영자가 볼 수 있는 간단 관리자 화면을 추가한 버전입니다.

추가 파일:

```text
admin.html
admin.css
admin.js
```

관리자 화면 주소:

```text
https://inhm1234.github.io/living_forest/admin.html
```

임시 비밀번호:

```text
forest2026
```

이 방식은 GitHub Pages에서 사용하는 프론트엔드 임시 비밀번호 방식입니다. 일반 방문자 메뉴에는 노출되지 않지만, 개발자 도구로 코드를 확인하면 비밀번호를 볼 수 있으므로 완전한 보안 방식은 아닙니다. 민감한 개인정보는 표시하지 않습니다.

이번 버전은 실제 GA4 데이터를 자동으로 불러오는 단계 전의 준비판입니다.
