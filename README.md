# 살아있는 숲

현재 테스트판: V1.21 test  
기준 버전: V1.20 test  
최근 업데이트: 오늘의 숲 문장 공유 1차  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.21 test 핵심 방향

V1.21 test는 V1.20에서 정리한 1차 완성 흐름을 다시 보강하는 버전이 아니라, 기록 후 생기는 `오늘의 숲 문장`을 밖으로 꺼내는 첫 확장판입니다.

목표 흐름은 다음과 같습니다.

1. 사용자가 오늘의 마음을 기록한다.
2. 숲 일기장에 오늘의 숲 문장이 남는다.
3. `오늘의 숲 나누기` 카드가 열린다.
4. 사용자는 오늘의 문장을 복사하거나 공유한다.
5. 공유 클릭은 기존 관리자 집계의 `share_click` 흐름으로 유지된다.

## 변경 내용

- 일반 화면 버전과 캐시값을 V1.21 test로 통일
- 감정 기록 후 `오늘의 숲 나누기` 카드 표시
- 오늘의 숲 문장 공유 미리보기 추가
- `문장 복사` 버튼 추가
- Web Share API를 지원하는 환경에서는 `공유하기` 사용
- Web Share API가 없으면 공유 문장 복사로 대체
- 기존 숲 일기장, 방문자 시스템, 관리자 / GA4 / Google Sheets / Apps Script 구조는 유지

## 이번 버전에서 하지 않은 것

- 숲 일기장 문구 추가 보강
- 월드 숲 비주얼 추가 보강
- 방문자 종류 추가
- 로그인 추가
- Google 계정 로그인 추가
- 서버 저장 추가
- 관리자 대시보드 고도화

## 확인 방법

1. GitHub에 V1.21 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.21 test인지 확인
4. 내 정원으로 들어가 감정 기록을 진행
5. 기록 후 `오늘의 숲 나누기` 카드가 보이는지 확인
6. `문장 복사`가 정상 작동하는지 확인
7. 모바일에서 지원되는 경우 `공유하기`가 공유창을 여는지 확인
8. 방문자 기록과 관리자 숫자가 기존처럼 유지되는지 확인

## 주의

- 이번 버전은 공유/확산 1차 확장판입니다.
- Apps Script를 새로 배포하지 않아도 됩니다.
- 기존 assets 폴더와 이미지 파일은 삭제하지 마세요.
