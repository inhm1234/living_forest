## 확인 기준

1. 상단 버전이 `V1.73.0 reset test`로 보이는지 확인
2. 내 정원이 노을이면 전체숲도 노을 배경으로 보이는지 확인
3. 내 정원이 밤이면 전체숲도 밤 배경으로 보이는지 확인
4. 전체숲에 V1.72.6의 주황/파랑 오버레이 느낌이 남지 않는지 확인
5. 내 정원 구조가 기존처럼 유지되는지 확인
