## V1.48.3 test 핵심 변경점

V1.48.3 test는 월드 숲 안의 A~E 자리를 실제 온라인 친구가 들어올 수 있는 고정 자리로 정의하고, 각 자리 이름과 위치를 확정한 버전입니다.

## 자리 이름

- A = 꽃길 자리
- B = 햇살 자리
- C = 연못 자리
- D = 그네 자리
- E = 꽃담 자리

## 수정한 것

- 버전 표기를 V1.48.3 test로 갱신
