# 살아있는 숲 V1.72.3 test

현재 V1.72.3 test는 **전체숲을 정원과 같은 파스텔 그림체를 유지한 전체숲 무대**로 다시 정리한 판입니다.

## 핵심 방향

- 전체숲은 빈 공터에 나무를 놓는 화면이 아니라, 숲길 주변에 여러 나무가 자연스럽게 자라난 장면처럼 보임
- 원형 자리/슬롯 느낌은 숨김
- 더미 나무는 배경 숲처럼 조용히 섞이고, 실제 친구 나무와 내 나무만 작게 식별 가능
- 하단 HUD는 `내 정원 / 내 자리 / 초대`만 유지
- 긴 설명 카드/관리 카드가 전체숲 아래에 노출되지 않게 유지
- 내 정원 V1.71.6의 고정 게임 무대 구조는 유지

## 확인 포인트

1. 상단 버전이 `V1.72.3 test`로 보이는지 확인
2. 전체숲이 공터/자리판처럼 보이지 않는지 확인
3. 나무들이 숲길 양옆과 깊이에 자연스럽게 섞여 보이는지 확인
4. 아래로 긴 스크롤이나 구형 설명 카드가 나오지 않는지 확인
5. `내 정원 / 내 자리 / 초대` 버튼만 보이는지 확인
