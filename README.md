# 살아있는 숲

현재 테스트판: V1.20 test  
기준 버전: V1.16.1 test  
최근 업데이트: 살아있는 숲 1차 완성형  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.20 test 핵심 방향

V1.20 test는 새 기능을 계속 덧붙이는 버전이 아니라, 지금까지 만든 기능을 하나의 1차 완성 흐름으로 정리한 버전입니다.

목표 흐름은 다음과 같습니다.

1. 월드 숲에서 서비스의 목적을 이해한다.
2. 내 정원으로 들어가 나무 이름을 정한다.
3. 하루 한 번 오늘의 마음을 고른다.
4. 내 나무가 자라고 오늘의 숲 문장이 남는다.
5. 숲 일기장에서 최근 기록을 확인한다.
6. 다음날 다시 들어왔을 때 이전 숲 문장 위에 오늘의 기록을 이어 쓴다.

## 변경 내용

- 일반 화면 버전과 캐시값을 V1.20 test로 통일
- 첫 화면 문구를 “나무 성장 + 숲 일기장” 중심으로 정리
- 숲 일기장 상태를 오늘 기록 전 / 오늘 기록 후로 구분
- 오늘 기록 전에는 어제 또는 이전 숲 문장을 먼저 보여주도록 정리
- 오늘 기록 후에는 오늘의 숲 문장과 최근 숲 기록이 명확하게 보이도록 정리
- 다음날 재방문 시 “이어쓰기” 느낌이 나도록 일일 흐름 문구 보강
- 기록은 이 기기에 저장된다는 안내 유지
- 관리자 / GA4 / Google Sheets / Apps Script / 방문자 시스템 구조는 유지

## 이번 버전에서 하지 않은 것

- 로그인 추가
- Google 계정 로그인 추가
- 서버 저장 추가
- 방문자 종류 추가
- 월드 숲 비주얼 대공사
- 관리자 대시보드 고도화

## 확인 방법

1. GitHub에 V1.20 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.20 test인지 확인
4. 내 정원으로 들어가 이름 저장 / 감정 기록을 확인
5. 숲 일기장에서 오늘 기록 전 / 기록 후 문구가 자연스럽게 바뀌는지 확인
6. 테스트 모드의 어제 기록 상태에서 이전 숲 문장 이어쓰기 흐름을 확인
7. 관리자 숫자와 방문자 시스템이 기존처럼 유지되는지 확인
