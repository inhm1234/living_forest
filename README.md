## V1.10.27 test 핵심 변경점

V1.10.27 test는 V1.10.26에서 확인된 문제인 **“내 나무 보기” 버튼이 눌러도 반응이 없어 보이는 문제**를 수정한 버전입니다.

## 원인

V1.10.22 이후 배경을 선명하게 만들기 위해 `ring`, `aura`, `glow`, `blur` 계열 효과를 대부분 제거했습니다.  
그 결과 `focusMyWorldSpot()` 함수는 실행되고 있었지만, 시각적으로 표시되는 강조 효과가 사라져서 버튼이 동작하지 않는 것처럼 보였습니다.

## 수정 내용

- 내 나무 보기 클릭 시 `focus_my_tree_click` 이벤트를 GA4로 전송
- 내 나무 보기 클릭 시 내 나무 위치로 부드럽게 스크롤
- 3초 정도 내 나무 크기/이름표/얇은 링만 살짝 강조
- 선명한 배경 구조와 GA4 행동 데이터 수집은 유지
- 다시 뿌옇게 만들던 overlay/blend 구조는 사용하지 않음
