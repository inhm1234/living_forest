# 살아있는 숲

현재 테스트판: V1.14 test  
기준 버전: V1.13 test  
최근 업데이트: 월드 숲 누적 변화 강화판  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.14 test 핵심 변경점

V1.14 test는 V1.13에서 만든 연속 기록 보상 흐름 다음 단계로, 누적 기록의 결과가 개인 정원 안에서만 끝나지 않고 월드 숲의 내 자리에서도 느껴지도록 개선한 버전입니다.

## 바뀐 점

- 월드 숲 첫 화면에 `월드 숲의 내 자리` 카드 추가
- 기록 일수에 따라 월드 숲 내 자리 상태 문구 강화
- 다음 월드 변화까지 남은 기록 수 표시
- 월드 숲 카드에 진행률 바 추가
- 내 나무 보기 버튼을 눌렀을 때 내 자리의 누적 성장 메시지 강화
- 성장 일수에 따라 내 자리 주변의 빛/존재감 단계 표현 보강

## 유지한 것

- 기존 localStorage 저장키 유지
- 관리자 대시보드 유지
- GA4 / Google Sheets / Apps Script 연결 유지
- 방문자 시스템 유지
- 나무 성장 이미지 유지
- assets 폴더는 포함하지 않음

## 확인 방법

1. GitHub에 V1.14 파일 업로드
2. 일반 화면 접속
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.14 test인지 확인
4. 월드 숲 첫 화면에 `월드 숲의 내 자리` 카드가 보이는지 확인
5. 테스트 모드에서 성장 일수를 바꿔가며 월드 숲 카드 문구와 진행률이 바뀌는지 확인
6. `내 나무 보기` 버튼을 눌렀을 때 내 자리 강조와 문구가 자연스러운지 확인
7. 감정 기록, 방문자, 관리자 숫자가 기존처럼 유지되는지 확인
