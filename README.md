# 살아있는 숲

현재 테스트판: **V1.8 test**  
기준 버전: **V1.7.1 test**  
최근 업데이트: 낮·노을·밤 월드 숲 분위기 강화  
공개 주소: https://inhm1234.github.io/living_forest/

## 프로젝트 소개

살아있는 숲은 하루에 한 번 지금의 마음을 남기면, 내 나무와 월드 숲이 조용히 자라는 웹 기반 체험판입니다.

현재 버전은 서버 저장이나 로그인 없이, 브라우저의 `localStorage`에 기록을 저장합니다. 그래서 기록은 이 기기와 이 브라우저에만 남습니다.

## 현재 포함된 기능

- 월드 숲 첫 화면
- 2.5D 월드 숲 레이어 구조
- 낮 / 노을 / 밤 시간대 분위기
- 내 나무 보기 강조 효과
- 개인 정원 이동
- 나무 이름 정하기
- 하루 1번 감정 기록
- 좋음 / 보통 / 피곤 감정에 따른 성장 반응
- 성장 목표 안내
- 월드 숲 내 자리 단계 안내
- 새 / 다람쥐 방문자 애니메이션
- 오늘 남은 방문자 흔적 카드
- 최근 방문자 기록 3개 표시
- 저장 구조 `schema 3` / `local-only` 표시
- 테스트 모드 `?test=1`

## V1.8 test 핵심 변경점

V1.8 test는 V1.7.1 test의 월드 숲 2.5D 구조를 기준으로, 시간대별 분위기를 더 확실하게 구분하는 테스트판입니다.

- 낮 숲: 햇빛, 밝은 숲길, 새와 나비 움직임 강화
- 노을 숲: 따뜻한 노을빛, 긴 그림자, 하루 마무리 감성 강화
- 밤 숲: 달빛, 안개, 반딧불이, 빛 입자 강화
- 월드 숲 요약 영역에 현재 시간대 분위기 안내 문구 추가
- 테스트 모드에서 낮 / 노을 / 밤 확인 링크 추가
- 생명체와 빛 입자 패턴이 약 20분 단위로 조금씩 달라지도록 보정

## 유지된 기능

- 기존 저장키 `livingForestV012` 유지
- 테스트 저장키 `livingForestV012_TEST` 유지
- 방문자 판정 저장 구조 유지
- 개인 정원 / 감정 기록 / 방문자 흔적 / 방문 기록 유지
- 서버 저장, 로그인, Google 계정 기능 없음
- 오늘의 운세코드 관련 기능 없음

## 테스트 방법

https://inhm1234.github.io/living_forest/

https://inhm1234.github.io/living_forest/?test=1

- 노을: https://inhm1234.github.io/living_forest/?test=1&worldTime=sunset
- 밤: https://inhm1234.github.io/living_forest/?test=1&worldTime=night
