# 살아있는 숲 V1.71.4 test

현재 V1.71.4 test는 **정원 화면을 모바일 게임처럼 고정된 무대로 다시 잡은 판**입니다.
하단 패널을 버리고, 정원 위에는 작은 HUD 버튼만 남겼습니다.
`기록 / 꾸미기 / 나눔`을 누르면 정원 화면을 밀어내지 않고 별도 팝업에서 기능이 열립니다.

## 핵심 방향

- 내 정원/내 숲 무대 크기는 버튼이나 기능 UI 때문에 줄거나 늘어나지 않음
- 긴 하단 패널, 큰 설명 바, 잘리는 메뉴 구조 제거
- 버튼은 작은 HUD처럼 정원 위에 얹고, 실제 기능은 팝업으로 실행
- 팝업 안 내용은 팝업 내부에서만 스크롤됨
- 완료 피드백은 정원 오른쪽 위 작은 로그로 표시
- 전체숲의 더미 나무 + 실제 친구 나무 표시 구조는 유지

## 확인 포인트

1. 상단 버전이 `V1.71.4 test`로 보이는지 확인
2. 정원 첫 화면에서 나무와 정원이 잘리지 않고 크게 보이는지 확인
3. 화면 아래에 긴 패널이 남아 있지 않은지 확인
4. `기록 / 꾸미기 / 나눔` 버튼이 작은 HUD처럼 보이는지 확인
5. 버튼을 누르면 정원을 밀지 않고 팝업이 열리는지 확인
6. 팝업을 닫으면 정원 화면이 그대로 돌아오는지 확인
7. 전체숲의 더미 나무와 실제 친구 나무 표시가 유지되는지 확인
