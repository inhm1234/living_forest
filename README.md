## V1.8.3 test 핵심 변경점

V1.8.3 test는 월드 숲의 낮 / 노을 / 밤 분위기가 내 정원에도 이어지도록 보정한 테스트판입니다.

- 낮 숲 → 낮 정원
- 노을 숲 → 노을 정원
- 밤 숲 → 밤 정원

특히 낮 정원은 기존의 고정된 어두운 밤 정원 느낌이 강했던 문제를 줄이기 위해 밝기와 시인성을 올리고, 반딧불이 같은 밤 연출은 약화했습니다.

## 유지되는 기능

- 개인 정원 이동
- 나무 이름 정하기
- 하루 1번 감정 기록
- 월드 숲 내 자리 반영
- 방문자 흔적 / 방문 기록
- 저장 구조 `schema 3` / `local-only`
- 테스트 모드 `?test=1`

## 테스트 방법

- 낮: `?test=1&worldTime=day`
- 노을: `?test=1&worldTime=sunset`
- 밤: `?test=1&worldTime=night`

확인할 것:
- 낮 숲에서 내 정원으로 가면 정원도 더 밝게 보이는지
- 노을 / 밤 정원도 자연스럽게 이어지는지
- 기존 기능과 기록이 유지되는지
