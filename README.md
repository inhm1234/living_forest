## V1.10.15 test 핵심 변경점

V1.10.15 test는 월드 숲이 단순히 비어 있는 초록 화면이 아니라, 사람이 실제로 바라보고 걸을 수 있을 것 같은 숲길과 그 양옆의 나무 구조로 느껴지도록 다듬은 테스트판입니다.

핵심 방향은 다음과 같습니다.
- 중앙에 사람이 지나갈 수 있을 것 같은 흙길/숲길을 더 분명하게 보이도록 조정
- 나무는 길 한가운데가 아니라, 길 양옆의 언덕과 잔디 자리에서 자라도록 재배치
- 바닥은 흙·잔디·풀 결이 함께 보이게 하여 “심겨 있는 느낌”을 강화
- 나무 수가 많지 않아도 길과 양옆 나무 구조 때문에 공간이 더 자연스럽게 느껴지도록 조정
- 가까운 나무와 먼 나무의 크기 차이를 더 분명하게 두어 원근감을 보정
- 개인 정원 성장, 방문자 흔적, 시간대 연동, 저장 구조는 유지

## 이번 버전에서 확인할 것

- 중앙에 사람이 걸을 수 있을 것 같은 길이 보이는지
- 길 양옆으로 나무가 심겨 있는 구조가 느껴지는지
- 땅 흙과 잔디, 작은 풀 결이 이전보다 더 잘 보이는지
- 나무가 허공에 떠 있는 것보다 실제로 심겨 있는 느낌이 드는지
- 가까운 나무/먼 나무의 차이로 원근감이 조금 더 느껴지는지
