## V1.36 test 핵심 변경점

V1.36 test는 V1.32~V1.36으로 나누어 진행할 수 있었던 다음 관리 확장들을 한 번에 묶은 통합본입니다.
반복 업로드 피로를 줄이기 위해, 새 기능들을 `내 숲 관리실` 흐름 아래에 모았습니다.

## 포함된 확장

- 내 숲 보관 파일 불러오기 / 복원
- 최근 14일 숲 달력
- 내 숲 기억 찾기 필터
- 내 숲 관리실 안내
- 선택형 백업 준비 안내

## 유지한 것

- 기존 내 숲 보관 파일 받기
- 요약 복사
- 숲 일기장
- 주간 숲 편지
- 오늘의 작은 돌봄
- 정원 표식
- 숲길 산책
- 숲의 소리
- 내일의 씨앗
- 나를 위한 작은 실천
- 배지 / 공유 / 초대장 / 방문자 시스템
- 관리자 / GA4 / Google Sheets / Apps Script 구조

## 주의

- 이번 버전은 서버 저장이나 Google 로그인 구현판이 아닙니다.
- 복원 기능은 선택한 보관 파일로 현재 이 기기의 숲 기록을 덮어쓸 수 있습니다.
- 복원 전에는 현재 숲을 먼저 `보관 파일 받기`로 내려받는 것을 추천합니다.
- assets 폴더는 ZIP에 포함되어 있지 않으므로 기존 assets 폴더를 삭제하지 마세요.
