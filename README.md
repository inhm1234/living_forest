# 살아있는 숲

현재 테스트판: V1.41 test  
기준 버전: V1.40 test  
최근 업데이트: 나무 중심 하단 패널 안정화  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.41 test 핵심 변경점

V1.41 test는 V1.40에서 적용한 “나무 중심 감성 UX/UI 전환”을 안정화한 버전입니다.
새로운 대형 기능을 늘리기보다, 사용자가 내 나무를 계속 보면서 기록 / 활동 / 꾸미기 / 편지 / 보관함을 오갈 수 있도록 하단 패널 구조의 실제 작동을 정리했습니다.

## 수정한 것

- 하단 패널 JS 변수 누락으로 앱 실행이 중간에 멈출 수 있던 문제 수정
- 기록 / 활동 / 꾸미기 / 편지 / 보관함 탭 전환 안정화
- 존재하지 않는 예전 진행 흐름 카드 코드와 CSS 정리
- V1.40 문서에 남아 있던 관리 확장 중심 설명을 V1.41 기준으로 정리
- index.html / app.js / style.css 캐시 버전을 V1.41 test로 통일
- assets 폴더 포함 업로드 기준으로 안내 문구 정리

## 유지한 것

- 월드 숲 / 내 정원 / 감정 기록
- 숲 일기장
- 주간 숲 편지
- 오늘의 작은 돌봄
- 정원 표식
- 숲길 산책
- 숲의 소리
- 내일의 씨앗
- 나를 위한 작은 실천
- 내 숲 보관 파일 받기 / 불러오기 / 복원
- 숲 달력 / 내 숲 기억 찾기
- 배지 / 공유 / 초대장 / 방문자 시스템
- 관리자 / GA4 / Google Sheets / Apps Script 구조

## 주의

- 이번 버전은 서버 저장이나 Google 로그인 구현판이 아닙니다.
- 복원 기능은 선택한 보관 파일로 현재 이 기기의 숲 기록을 덮어쓸 수 있습니다.
- 복원 전에는 현재 숲을 먼저 `보관 파일 받기`로 내려받는 것을 추천합니다.
- `assets` 폴더는 포함되어 있으므로 저장소 루트의 `assets/` 구조를 유지해 업로드하세요.
