## V1.49 test 핵심 변경점

V1.49 test는 월드 숲의 꽃길 / 햇살 / 연못 / 그네 / 꽃담 자리를 단순 표시가 아니라 실제 초대 흐름으로 연결하는 버전입니다.

## 수정한 것

- 버전 표기를 V1.49 test로 갱신
- 월드 숲 안의 자리 배지를 클릭 가능한 버튼으로 변경
- 친구 초대 카드에 자리 선택 버튼 5개 추가
- 선택한 자리별로 초대 문구가 달라지도록 수정
- 초대 링크에 `seat` 파라미터 추가
- 선택한 자리 표시 / 복사 완료 문구를 자리별로 표시
- 나중에 `꽃길 자리 · 친구 닉네임` 구조로 확장하기 쉬운 형태로 정리
