# 살아있는 숲

현재 검증판: **V1.58 test**

## V1.58 test 핵심 정리

V1.58 test는 **첫 방문 UX/UI 개편판**입니다.  
블로그나 검색으로 처음 들어온 사용자가 긴 설명을 읽지 않아도 “무엇을 해야 하는지” 바로 이해하도록 첫 화면과 안내 흐름을 정리했습니다.

### 이번 버전에서 바뀐 점

- 첫 화면 상단에 **처음 온 분을 위한 시작 안내** 카드 추가
- 가장 큰 행동 버튼을 **오늘의 기록 시작하기**로 정리
- 첫 방문자용 **30초 온보딩 모달** 추가
- 정원 화면에 **나무 이름 → 마음 선택 → 변화 확인** 안내 카드 추가
- 글씨 대비, 카드 배경, 버튼 색감을 더 밝고 읽기 쉽게 보정
- 기존 카카오 초대, 친구 자리, 방문자, 보관함 기능은 유지

## 출시 방향 메모

이 프로젝트의 현재 목표는 지인 홍보형 테스트가 아니라,  
**블로그 소개글 / 선택적 유튜브 영상 / 검색 유입**으로 처음 온 사람이 혼자 이해하고 첫 행동을 할 수 있는 MVP입니다.

그래서 이번 버전의 우선순위는 기능 추가보다 다음 기준에 있습니다.

1. 처음 온 사람이 10초 안에 무엇을 해야 하는지 알기
2. 글씨가 잘 보이고 버튼이 명확하기
3. 타겟층에 맞는 귀엽고 밝은 UX/UI로 보이기
4. 첫 행동인 “오늘의 기록”까지 자연스럽게 이동하기
