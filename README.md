# 살아있는 숲

현재 테스트판: V1.25.2 test  
기준 버전: V1.25.1 test  
최근 업데이트: 오늘의 작은 돌봄 버튼 위치 보정  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.25.2 test 핵심 변경점

V1.25.2 test는 V1.24에서 추가한 `오늘의 작은 돌봄`이 화면 아래 카드에만 있어서 사용자가 정원 무대의 변화를 바로 확인하기 어려웠던 문제를 보정한 버전입니다.

## 변경 내용

- 감정 기록 후 정원 무대 안, 나무 가까운 위치에 `오늘의 작은 돌봄` 빠른 선택 패널 추가
- 물 / 빛 / 쉼 버튼을 정원 화면 안에서 바로 선택 가능하게 변경
- 선택 후 패널에 오늘의 돌봄 결과가 바로 표시되도록 보정
- 물 주기 / 빛 쬐기 / 쉬게 하기 선택에 따라 나무 주변 빛 효과가 달라지도록 보강
- 기존 아래쪽 `오늘의 작은 돌봄` 카드도 유지하여 상세 문구 확인 가능
- 관리자 / GA4 / Google Sheets / Apps Script / 방문자 시스템 / 서버 저장 구조는 유지

## 확인 방법

1. GitHub에 V1.25.2 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.25.2 test인지 확인
4. 내 정원으로 이동
5. 감정 기록 후 나무 가까이에 `오늘의 작은 돌봄` 빠른 선택 패널이 보이는지 확인
6. 물 / 빛 / 쉼 중 하나를 눌렀을 때 바로 선택 상태와 나무 주변 빛 변화가 보이는지 확인
7. 아래쪽 돌봄 카드에도 선택 결과가 반영되는지 확인
8. 내 정원 표식, 숲 일기장, 공유, 초대장, 배지 기능이 기존처럼 유지되는지 확인

## 주의

- 이번 버전은 새 확장이 아니라 V1.24/V1.25 상호작용 UX 보정판입니다.
- 로그인, 서버 저장, Google 계정 로그인은 추가하지 않았습니다.
- Apps Script를 새로 배포하지 않아도 됩니다.
- 기존 assets 폴더와 이미지 파일은 삭제하지 마세요.
