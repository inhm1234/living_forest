## V1.53 test 핵심 변경점

초대 링크로 들어온 친구가 이미 이 브라우저에서 나무를 키우고 있다면, 새 나무를 만들지 않고 기존 나무 상태로 초대받은 자리에 들어갈 수 있게 했습니다.

## 수정한 것

- 초대받은 친구 화면에 “이미 내 나무가 있어요” 영역 추가
- 기존 나무 이름 / 성장일 / 성장 단계 / 최근 마음 요약 표시
- “기존 내 나무로 이 자리에 들어가기” 버튼 추가
- 처음 온 친구는 기존처럼 새 나무를 만들 수 있음
- 친구 자리 저장소에는 기존 나무의 성장 상태가 저장됨

## 중요
