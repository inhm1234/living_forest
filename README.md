# 살아있는 숲

현재 테스트판: V1.27.1 test  
기준 버전: V1.26 test  
최근 업데이트: 숲의 소리 버튼 수정판  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.27.1 test 핵심 변경점

V1.27.1 test는 V1.20 1차 완성형 이후의 추가 확장 흐름입니다.  
V1.26의 숲길 산책을 다시 보강하지 않고, 내 정원에 짧은 배경 소리를 켜는 **감각/몰입 확장**을 추가했습니다.

## 변경 내용

- 내 정원 화면에 `숲의 소리` 카드 추가
- 바람 / 물방울 / 밤벌레 소리 선택 기능 추가
- 별도 음원 파일 없이 브라우저 Web Audio로 짧은 분위기 소리 생성
- `끄기` 버튼으로 즉시 정지 가능
- 화면을 떠나면 소리가 자동으로 멈추도록 처리
- `forest_sound_played` 이벤트 추적 추가
- 로그인 / 서버 저장 / 관리자 구조 / Apps Script / Google Sheets 구조는 유지

## 확인 방법

1. GitHub에 V1.27 파일 업로드
2. 일반 화면 접속
3. 화면 상단이 V1.27.1 test인지 확인
4. 내 정원에 `숲의 소리` 카드가 보이는지 확인
5. 바람 / 물방울 / 밤벌레 버튼을 눌렀을 때 소리가 재생되는지 확인
6. 끄기 버튼 또는 화면 이동 시 소리가 멈추는지 확인
7. 기존 숲 일기장, 공유, 초대장, 배지, 돌봄, 표식, 산책 기능이 유지되는지 확인

## 주의

- 이번 버전은 감각/몰입 확장입니다.
- 외부 음원 파일이나 assets 추가가 필요 없습니다.
- 일부 브라우저에서는 사용자 클릭 후에만 소리가 재생됩니다.


## V1.27.1 test 수정 내용

- V1.27에서 누락된 숲의 소리 버튼 클릭 이벤트 연결
- 바람 / 물방울 / 밤벌레 버튼 동작 복구
- 끄기 버튼 동작 복구
- 전체 숲 이동 또는 화면 숨김 시 소리 자동 정지

이번 버전은 새 확장이 아니라 V1.27 숲의 소리 기능의 동작 수정판입니다.
