# 살아있는 숲

현재 테스트판: **V1.7.1 test**  
기준 버전: **V1.7 test**  
최근 업데이트: 월드 숲 2.5D 세부 보정  
공개 주소: https://inhm1234.github.io/living_forest/

## 프로젝트 소개

살아있는 숲은 하루에 한 번 지금의 마음을 남기면, 내 나무와 월드 숲이 조용히 자라는 웹 기반 체험판입니다.

현재 버전은 서버 저장이나 로그인 없이, 브라우저의 `localStorage`에 기록을 저장합니다. 그래서 기록은 이 기기와 이 브라우저에만 남습니다.

## 현재 포함된 기능

- 월드 숲 첫 화면
- 2.5D 월드 숲 레이어 구조
- 낮 / 노을 / 밤 분위기 기초
- 개인 정원 이동
- 내 나무 보기 강조 효과
- 나무 이름 정하기
- 하루 1번 감정 기록
- 좋음 / 보통 / 피곤 감정에 따른 성장 반응
- 3일 / 7일 / 14일 / 21일 / 30일 성장 목표 안내
- 월드 숲 내 자리 단계 안내
- 새 / 다람쥐 방문자 애니메이션
- 오늘 남은 방문자 흔적 카드
- 최근 방문자 기록 3개 표시
- 저장 구조 `schema 3` / `local-only` 표시
- 테스트 모드 `?test=1`

## V1.7.1 test 핵심 변경점

V1.7.1 test는 V1.7 test의 월드 숲 2.5D 화면을 한 번 더 보기 편하게 다듬은 세부 보정판입니다.

- 월드 숲 전체 어두움과 하단 그림자 강도 완화
- 중앙 숲길과 빈터가 더 잘 보이도록 밝기와 대비 보정
- 내 나무 자리 카드 크기와 투명도 조정
- 내 나무 이름/상태 문구 가독성 개선
- 주변 자리 이름표 가독성 개선
- 새/다람쥐/나비/반딧불이/빛 입자 등장 수와 투명도 보정
- 너무 많은 생명체가 동시에 보이지 않도록 조용한 순간 확률 추가
- 테스트 모드에서 `worldTime=day/sunset/night`로 시간대별 분위기 확인 가능

## V1.7 test에서 유지된 방향

- 첫 화면을 단순 배경이 아니라 관람용 월드 숲처럼 구성
- 앞쪽 풀/잎/안개, 중간 숲길/빈터/내 자리, 뒤쪽 깊은 숲 레이어 유지
- 낮/노을/밤 3단계 분위기 기초 유지
- 생명체 움직임 기초 구조 유지
- 내 나무 보기 강조 버튼 유지

## V1.5~V1.6에서 유지된 저장/방문자 구조

- 저장 방식 정보를 `storageInfo`로 분리
- `dataSchemaVersion` 3 유지
- 현재 저장 방식은 `local-only` / `browser-localStorage`
- 추후 서버 저장용 자리 `serverUserId`, `serverTreeId`, `lastSyncedAt` 유지
- 기존 저장키 `livingForestV012` 유지
- 테스트 저장키 `livingForestV012_TEST` 유지
- 방문자 판정 저장 구조 유지
- 오늘 기록 전에는 방문자가 자동 등장하지 않음
- 오늘 감정 기록 후 확률에 따라 방문자 판정
- 이미 오늘 기록한 사용자도, 오늘 방문자 판정이 없으면 개인 정원 입장 시 하루 1회 판정
- 오늘 방문자 판정이 이미 있으면 새로고침해도 다시 뽑지 않음

## 테스트 방법

일반 화면:

https://inhm1234.github.io/living_forest/

테스트 모드:

https://inhm1234.github.io/living_forest/?test=1

시간대별 월드 숲 테스트:

https://inhm1234.github.io/living_forest/?test=1&worldTime=day
https://inhm1234.github.io/living_forest/?test=1&worldTime=sunset
https://inhm1234.github.io/living_forest/?test=1&worldTime=night

테스트 모드에서 확인할 것:

- 데이터 표시에 `V1.7.1 test · schema 3 · local-only`가 보이는지
- 월드 숲이 V1.7보다 조금 더 보기 편하고 덜 답답하게 보이는지
- 중앙 숲길/빈터가 너무 묻히지 않는지
- 내 나무 자리 카드가 너무 크거나 답답하게 느껴지지 않는지
- 내 나무 보기 버튼의 강조 효과가 자연스러운지
- 새/다람쥐/나비/반딧불이/빛 입자가 너무 많거나 산만하지 않은지
- 개인 정원, 감정 기록, 방문자 흔적, 방문 기록이 기존처럼 유지되는지
- 기존 나무 이름과 성장 기록이 이어지는지
