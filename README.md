## V1.10.9 test 핵심 변경점

V1.10.9 test는 월드 숲이 이미 완성된 큰 배경 숲처럼 보이고, 그 위에 사용자가 키운 나무를 작게 뿌리는 방식이 어색했던 문제를 줄이기 위한 테스트판입니다.

핵심 방향은 다음과 같습니다.
- 거대한 배경 나무 느낌을 크게 줄이고, 열린 언덕/빈터 같은 월드 구조로 전환
- 월드 숲의 중심을 배경 이미지가 아니라 여러 줄의 숲자리와 사용자 나무로 이동
- 비어 있는 자리와 채워진 나무를 함께 보여주어 야구장 관중석처럼 점점 채워지는 감각 테스트
- 내 나무는 공동 숲의 한 자리로 보이되, 내 나무 보기 버튼을 누르면 은은하게 찾아주는 구조 유지
- 개인 정원 성장, 나무 성장 이미지, 방문자 흔적, 낮/노을/밤 시간대 연동은 유지

## 이번 버전에서 확인할 것

- 배경 숲이 더 이상 이미 완성된 거대한 숲처럼 보이지 않는지
- 비어 있는 자리와 채워진 나무들이 “관중석처럼” 줄지어 보이는지
- 내 나무가 많은 자리 중 하나를 채웠다는 느낌이 나는지
- 가까이서는 개별 나무, 멀리서는 공동 숲처럼 느껴지는지
- 나무들이 너무 작거나 장식처럼 보이지 않는지
