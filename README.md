# 살아있는 숲

현재 검증판: **V1.64 test**

## V1.64 test 핵심 정리

V1.60 test는 **첫 기록 흐름 개선판**입니다.  
V1.59에서 온보딩 모달은 통과했고, 이번 버전은 “내 숲 시작하기”를 누른 뒤 사용자가 바로 나무 이름 입력과 마음 선택을 찾을 수 있도록 정원 화면의 첫 기록 동선을 다시 정리했습니다.

### 이번 버전에서 바뀐 점

- 정원 화면에서 **기록 패널을 숲 이미지보다 먼저 보이도록 위치 조정**
- “오늘의 기록 시작하기” / “내 숲 시작하기” 클릭 시 **기록 패널 자동 열림**
- 첫 방문자는 **나무 이름 입력칸으로 자동 스크롤 및 포커스**
- 나무 이름이 이미 있으면 **마음 선택 카드로 이동**
- 첫 기록 패널에 부드러운 강조 효과 추가
- 입력칸/마음 선택 버튼의 크기와 대비 보완
- 기존 카카오 초대, 친구 자리, 방문자, 보관함 기능은 유지

## 출시 방향 메모

이 프로젝트의 현재 목표는 지인 홍보형 테스트가 아니라,  
**블로그 소개글 / 선택적 유튜브 영상 / 검색 유입**으로 처음 온 사람이 혼자 이해하고 첫 행동을 할 수 있는 MVP입니다.

그래서 이번 버전의 우선순위는 기능 추가보다 다음 기준에 있습니다.

1. 처음 온 사람이 10초 안에 무엇을 해야 하는지 알기
2. 글씨가 잘 보이고 버튼이 명확하기
3. 타겟층에 맞는 귀엽고 밝은 UX/UI로 보이기
4. 첫 행동인 “오늘의 기록”까지 자연스럽게 이동하기

## 업로드 후 확인 체크리스트

1. 상단 버전이 `V1.64 test`로 보이는지 확인
2. 첫 방문 상태에서 30초 안내 모달이 뜨는지 확인
3. 온보딩 뒤의 숲 배경이 너무 뭉개지지 않고 분위기가 보이는지 확인
4. “내 숲 시작하기” 버튼이 가장 눈에 띄는지 확인
5. “오늘의 기록 시작하기” 버튼을 누르면 내 정원 화면으로 이동하고 기록 패널이 자동으로 열리는지 확인
6. 정원 화면에서 나무 이름 입력칸과 마음 선택 버튼이 숲 이미지보다 먼저 보이는지 확인
7. 카카오 초대/친구 자리 기능이 기존처럼 유지되는지 확인
