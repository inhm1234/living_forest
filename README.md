공개 주소: https://inhm1234.github.io/living_forest/

## V1.10.12 test 핵심 변경점

V1.10.12 test는 현재 월드 숲의 나무 수가 많지 않아도, 공간이 더 넓고 깊이 있게 느껴지도록 시점과 원근감을 보정한 테스트판입니다.

핵심 방향은 다음과 같습니다.
- 월드 숲에 멀리 보기 / 가까이 보기 버튼 추가
- 멀리 보기에서는 넓은 월드 숲과 전체 자리 흐름을 보도록 조정
- 가까이 보기에서는 내 나무 주변 자리와 앞쪽 숲결이 조금 더 확대되어 보이도록 조정
- 앞/중간/뒤 레이어의 크기와 흐림 차이를 더 줘서 원근감 강화
- 나무 수가 적어도 “앞으로 채워질 숲”처럼 느껴지도록 유지
- 개인 정원 성장, 나무 성장 이미지, 방문자 흔적, 낮/노을/밤 시간대 연동은 유지

## 이번 버전에서 확인할 것

- 멀리 보기에서 월드 숲 전체 흐름이 더 잘 보이는지
- 가까이 보기에서 내 나무 주변과 앞쪽 자리가 더 잘 느껴지는지
- 같은 나무 수라도 이전보다 원근감이 더 느껴지는지
- 멀리/가까이 보기 전환이 너무 어색하지 않은지
- 나무 수가 적은 상태여도 허전하기보다 “채워져 갈 숲”처럼 느껴지는지
