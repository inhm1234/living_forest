## V1.10.5 test 핵심 변경점

V1.10.5 test는 월드 숲이 숲 속 무대처럼 보이던 느낌을 줄이고,
중앙 밝기 / 배치 규칙성 / 전경 가림을 보정해 더 자연스럽고 불규칙한 숲 장면으로 보이도록 다듬은 테스트판입니다.

## 이번 버전에서 확인할 것

- 중앙 빈터가 전보다 덜 무대처럼 보이는지
- 나무들이 조금 더 불규칙하고 자연스럽게 배치된 것처럼 느껴지는지
- 나무 하단이 전경과 자연스럽게 묻히는지
- 내 나무가 주인공이되 과한 전시 느낌은 줄었는지
