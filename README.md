# 살아있는 숲

현재 테스트판: V1.15 test  
기준 버전: V1.14.1 test  
최근 업데이트: 월드 숲 자리감 강화판  
공개 주소: https://inhm1234.github.io/living_forest/

## V1.15 test 핵심 변경점

V1.15 test는 V1.14.1에서 정상화한 `월드 숲의 내 자리` 흐름 다음 단계로, 내 나무가 월드 숲에 단순히 표시되는 수준을 넘어 실제 숲 안의 한 자리에 심겨 있는 것처럼 느껴지도록 보강한 버전입니다.

## 변경 내용

- 월드 숲 안에 비어 있는 자리 빛 표시 추가
- 내 자리 주변 바닥빛과 링 효과 강화
- 내 나무 이름표 가독성 보강
- 주변 나무들이 숲의 자리처럼 느껴지도록 라벨/깊이감 보정
- `월드 숲 자리 안내` 카드 추가
- `내 나무 보기` 버튼 클릭 시 내 자리 포커스 효과 강화
- 관리자 / GA4 / Google Sheets / Apps Script 구조는 유지

## 확인 방법

1. GitHub에 V1.15 파일 업로드
2. 일반 화면 접속  
   https://inhm1234.github.io/living_forest/
3. 화면 상단 버전이 V1.15 test인지 확인
4. 월드 숲에 비어 있는 자리 빛과 내 자리 강조가 보이는지 확인
5. `월드 숲 자리 안내` 카드가 자연스럽게 보이는지 확인
6. `내 나무 보기` 버튼을 눌렀을 때 내 자리가 더 잘 드러나는지 확인
7. 내 정원 이동, 감정 기록, 관리자 숫자 표시가 기존처럼 유지되는지 확인

## 주의

- 이번 버전은 프론트 화면 개선판입니다.
- Apps Script를 새로 배포하지 않아도 됩니다.
- 기존 assets 폴더와 이미지 파일은 삭제하지 마세요.
