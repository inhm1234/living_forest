오늘의 운세코드와 섞지 말 것

## V1.57 test 핵심 정리

V1.57 test는 V1.56 test에서 정리한 친구 배치 UX를 더 안전하게 만드는 안정화 패치입니다.  
저장소 구조는 바꾸지 않고, 친구를 자리로 이동하거나 이미 친구가 있는 자리와 교체할 때 실수와 중복 클릭을 줄이는 것이 목표입니다.

## 이번 버전에서 정리한 것

- 친구 배치 저장 중에는 다른 자리 버튼을 다시 누르지 못하게 방지
- 현재 자리 버튼은 눌리지 않도록 잠금 처리
- 빈 자리 배치 / 다른 빈 자리로 이동 / 친구가 있는 자리와 교체 상태를 더 명확히 구분
- 이동 버튼을 별도 상태로 표시
- 교체 확인창에 이동 전 자리와 이동 후 자리를 함께 표시
- 교체 취소 시 “변화 없음” 안내 메시지 표시
- 저장 성공 메시지를 배치/이동/교체 상황에 맞게 정리
- 홈페이지 상단 버전, README, VERSION.txt를 `V1.57 test`로 통일

## 현재까지 완료된 안정 기준

## Apps Script 안내

이번 V1.57 test는 프론트 화면 안정화 패치입니다.  
Apps Script 코드는 V1.55 stable에서 배포한 `list_friend_links`, `assign_friend_seat`가 그대로 있으면 됩니다.

즉, 이번 버전은 Apps Script 새 배포가 필요 없습니다.
