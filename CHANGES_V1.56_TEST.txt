   - 다른 친구가 있는 자리는 `교체`
   - 버튼 안에 짧은 안내 문구 추가

3. 친구 목록 정렬 개선
   - 이미 자리에 배치된 친구가 먼저 보이도록 정리
   - 아직 자리 없는 친구는 뒤쪽에 표시

4. 교체 확인창 문구 개선
   - 누가 빠지고 누가 들어가는지 더 쉽게 설명
   - 교체된 친구가 삭제되지 않는다는 점을 명확히 표시
