오늘의숲 나무 동물 부르기 — 운영 반영본

업로드 위치: GitHub 저장소 루트
덮어쓸 파일:
- index.html
- app.js
- tree-animal-call.js
- tree-animal-call.css

Supabase SQL은 이미 설치 및 검증했으므로 다시 실행하지 않습니다.
DEV에서 확인한 최신 수정 포함:
- 나무 클릭 영역 pointer-events 활성화
- 3회 유효 클릭 / 3초 후 동물 등장
- 1시간 쿨타임 및 새로고침 후 유지
- 쿨타임 문구를 정원 카드 위에 선명하게 표시
- 캐시 버전 갱신

배포 후 운영 주소에서 강력 새로고침(Ctrl+Shift+R) 후 1회만 실제 호출 테스트합니다.
