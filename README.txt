[검수 순서]
1. 완성된 공유나무 상세 진입
2. '이 나무에 한마디를 남겨볼까요?' 영역 표시 확인
3. A 계정에서 1~40자 문장 저장
4. 저장 문장이 '나의 한마디'에 표시되는지 확인
5. 새로고침 후에도 유지되는지 확인
6. 같은 계정에서 문장을 수정하고 반영되는지 확인
7. B 계정에서 A 문장이 친구의 한마디로 바로 보이는지 확인
8. B 계정도 문장을 남기면 두 문장이 함께 보이는지 확인
9. 진행 중 공유나무에는 한마디 영역이 보이지 않는지 확인
10. 모바일에서 카드·입력창·버튼이 잘리지 않는지 확인
