이번 범위
- 완성된 공유나무를 보관한 채 같은 친구와 다음 나무 제안 가능
- 상대가 수락하면 새 진행 중 공유나무 생성
- 한 친구 사이 진행 중 나무는 동시에 1그루만 허용
- 완성된 이전 나무는 함께한 숲에 그대로 유지
- 제안 대기/도착 상태를 친구 카드와 함께한 숲에서 표시

SQL 재실행
- 이 패키지의 SQL은 1회 실행 필요
- 기존 완성 나무와 빛 기록은 삭제하지 않음
