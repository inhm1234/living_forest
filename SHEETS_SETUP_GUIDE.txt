살아있는 숲 V1.12 관리자 데이터 안내

이번 V1.12는 관리자 데이터 구조를 새로 바꾸는 버전이 아닙니다.

이미 V1.10.31에서 적용한 날짜별 집계용 Apps Script가 정상 작동 중이라면 그대로 사용하면 됩니다.

확인할 것:
1. 관리자 화면 접속
   https://inhm1234.github.io/living_forest/admin.html

2. 비밀번호 입력
   forest2026

3. 오늘 / 어제 / 최근 7일 / 전체 선택이 계속 보이는지 확인

4. 일반 화면에서 내 정원 이동, 감정 기록 후 관리자 숫자가 계속 찍히는지 확인

이번 V1.12 업로드 때 Apps Script를 새로 배포하지 않아도 됩니다.

이번 V1.12의 목적은 다음날 다시 왔을 때 어제의 기록과 오늘의 기록이 이어지는 느낌을 강화하는 것입니다.
