살아있는 숲 V1.14 관리자 데이터 안내

이번 V1.14는 관리자 데이터 구조를 새로 바꾸는 버전이 아닙니다.

기존에 V1.10.31 이후 적용한 Apps Script / Google Sheets 연결을 그대로 사용합니다.

유지되는 항목:
- Google Sheets events 시트
- Apps Script summary 집계
- 관리자 화면 날짜 기준 선택
- 오늘 / 어제 / 최근 7일 / 전체 집계
- app.js 이벤트 전송 구조
- admin.js 데이터 불러오기 구조

따라서 이번 V1.14 업로드 때 Apps Script를 새로 배포하지 않아도 됩니다.

이번 V1.14의 목적은 누적 기록의 결과가 월드 숲의 내 자리에서도 느껴지게 만드는 것입니다.
