살아있는 숲 V1.13 관리자 데이터 안내

이번 V1.13은 관리자 데이터 구조를 새로 바꾸는 버전이 아닙니다.

유지되는 것:
- GA4 측정 ID G-YC872G7MH1
- Google Sheets events 기록
- Apps Script summary 집계
- 관리자 페이지 admin.html
- 날짜 기준 선택: 오늘 / 어제 / 최근 7일 / 전체

이번 버전에서 확인할 것:
1. 일반 페이지 접속 후 app_opened가 기록되는지
2. 내 나무 키우러 가기 클릭 후 go_garden_click이 기록되는지
3. 감정 기록 후 mood_recorded가 기록되는지
4. 관리자 화면에서 기존 숫자가 계속 보이는지

이번 V1.13 업로드 때 Apps Script를 새로 배포하지 않아도 됩니다.

이번 V1.13의 목적은 연속 기록을 이어갈 때 다음 보상과 작은 변화를 기다리게 만드는 것입니다.
