살아있는 숲 V1.11 관리자 데이터 안내

이번 V1.11은 관리자 데이터 구조를 새로 바꾸는 버전이 아닙니다.

기준:
- V1.10.31에서 만든 날짜별 집계 구조 유지
- 오늘 / 어제 / 최근 7일 / 전체 선택 유지
- Apps Script 웹 앱 URL 유지
- Google Sheets events 시트 기록 유지

따라서 V1.10.31 이후 관리자 숫자가 정상 표시되고 있다면,
이번 V1.11 업로드 때 Apps Script를 새로 배포하지 않아도 됩니다.

이번 V1.11의 목적은 감정 기록 후 사용자가 “내일 다시 와야 이어진다”는 기대감을 느끼게 만드는 것입니다.
