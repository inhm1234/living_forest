V1.50 온라인 친구 저장소 설정 가이드

목표:
GitHub Pages는 정적 사이트라서 친구 데이터를 직접 저장할 수 없습니다.
그래서 Google Apps Script + Google Sheets를 공통 저장소로 사용합니다.

1. Google Apps Script 열기
- 기존 살아있는 숲 Apps Script 프로젝트를 엽니다.
- 기존 코드 전체를 APPS_SCRIPT_CODE.txt 내용으로 교체합니다.

2. 저장 후 배포
- 저장합니다.
- 배포 > 배포 관리로 이동합니다.
- 웹 앱 배포를 새 버전으로 업데이트합니다.
- 실행 권한은 기존처럼 본인 계정, 접근 권한은 웹 앱 접근 가능 상태를 유지합니다.

3. GitHub 업로드
- index.html, style.css, app.js 등을 V1.50 패치 파일로 덮어 업로드합니다.

4. 테스트 방법
- 내 화면에서 꽃길/햇살/연못/그네/꽃담 자리 중 하나를 선택합니다.
- 문구 미리보기를 눌러 링크를 확인합니다.
- 해당 링크를 시크릿 창 또는 다른 브라우저에서 엽니다.
- 온라인 친구 자리 초대 카드가 뜨는지 확인합니다.
- 닉네임과 나무 이름을 입력합니다.
- Google Sheets에 friend_seats 시트가 생기고 데이터가 저장되는지 확인합니다.
- 원래 브라우저에서 새로고침하면 해당 자리에 친구 이름이 표시되는지 확인합니다.

주의:
- Apps Script를 배포하지 않으면 저장/불러오기는 실패합니다.
- 실패 메시지가 나오면 배포 권한, 웹 앱 URL, 스프레드시트 ID를 확인해야 합니다.
