오늘의숲 첫 방문 튜토리얼 v1 (DEV 전용)

업로드할 파일
- dev/garden-login-test/index.html
- dev/garden-login-test/garden.js
- dev/garden-login-test/garden.css

이번 수정 범위
- 운영 루트 파일은 수정하지 않았습니다.
- Supabase SQL/테이블 변경은 없습니다.
- 로그인, 기록 저장, 동물, 편지, 친구, 공유나무, 꾸미기 기능의 규칙은 변경하지 않았습니다.

첫 방문 흐름
1. 나무 이름을 정한 뒤, 기록이 0개인 계정에서 정원을 2.3초 보여줍니다.
2. "여기는 오늘의 마음이 머무는 작은 정원이에요."가 나타납니다.
3. "오늘의 마음 하나를 나무에 남겨볼까요?"로 바뀌고 마음 남기기 버튼이 은은하게 강조됩니다.
4. 마음 남기기를 누르면 기록 시트 안에 "정답은 없어요..." 안내 한 줄만 보입니다.
5. 첫 기록이 끝나면 기존 나무 반응/발견 반짝임을 유지하면서, 반짝임을 찾도록 한 줄로 안내합니다.
6. 작은 것을 찾으면 안내는 자동으로 끝납니다.

검수 방법
- 실제 흐름: 기록이 없는 DEV 계정에서 로그인하면 자동 시작합니다.
- 화면 연출만 보기: DEV 주소 뒤에 ?tutorialPreview=intro 를 붙이면 첫 안내 후 버튼 강조 장면을 볼 수 있습니다.
  예: /dev/garden-login-test/?tutorialPreview=intro
- 기존 기록/동물/편지 데이터는 바꾸지 않습니다.

검수 포인트
- 첫 화면이 설명 카드로 가득 차지 않는지
- 마음 남기기 버튼을 자연스럽게 찾을 수 있는지
- 기록 시트에서 문구가 부담스럽지 않은지
- 첫 기록 뒤 작은 반짝임을 찾는 흐름이 재미있는지
