살아있는 숲 V1.41.1 test — 하단 패널 가림 보정판

목적:
- V1.41에서 하단 패널은 작동하게 되었지만, 첫 진입 화면에서 패널이 숲 무대를 너무 많이 차지하는 문제가 있었습니다.
- 사용자가 내 정원에 들어왔을 때 먼저 나무/숲 무대를 보고, 필요할 때만 기록/활동/꾸미기/편지/보관함을 펼칠 수 있도록 보정했습니다.

수정 내용:
1. 내 정원 첫 진입 시 패널 본문을 기본 접힘 상태로 변경
2. 접힌 상태에서는 메뉴 탭과 패널 열기 버튼만 얇게 표시
3. 탭을 누르면 해당 탭 내용이 펼쳐지도록 유지
4. 패널 접기 버튼을 패널 열기/접기 토글 버튼으로 변경
5. 펼친 상태의 패널 최대 높이를 줄여 나무와 하단 메시지가 덜 가려지도록 조정
6. 정원 무대 높이를 약간 확보해 첫 화면에서 숲 비주얼이 더 잘 보이도록 보정

유지한 것:
- 감정 기록 구조
- 방문자 시스템
- 내 숲 보관함 / 복원 / 숲 달력 / 기억 찾기
- 관리자 / GA4 / Google Sheets / Apps Script
- localStorage 저장 키
- assets 파일 구조

1. 일반 화면 상단이 V1.41.1 test인지 확인

3. 숲/나무 무대가 이전보다 더 넓게 보이는지 확인
4. 기록 / 활동 / 꾸미기 / 편지 / 보관함 탭을 눌렀을 때 패널이 펼쳐지는지 확인
5. 패널 접기 버튼으로 다시 얇게 접히는지 확인
