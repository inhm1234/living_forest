살아있는 숲 V1.1 안정판 파일 적용 안내

이 ZIP은 전체 프로젝트가 아니라, GitHub에 덮어쓸 안정판 파일 묶음입니다.

덮어쓸 파일:
- index.html
- style.css
- app.js
- VERSION.txt
- LAUNCH_CHECKLIST.txt
- PUBLIC_SHARE_COPY.txt
- UPLOAD_GUIDE.txt

중요:
- assets 폴더는 삭제하지 마세요.
- assets/garden, assets/world, assets/share 폴더는 그대로 유지해야 이미지가 깨지지 않습니다.
- 서버 저장, 로그인, Google 계정 기능은 들어 있지 않습니다.
- 일반 저장키 livingForestV012와 테스트 저장키 livingForestV012_TEST는 그대로 유지됩니다.
- V1.1은 V1.1 test에서 정상 확인된 성장 목표 안내를 안정판으로 확정한 버전입니다.

GitHub 적용 순서:
1. GitHub 저장소 inhm1234/living_forest로 이동합니다.
2. 위 파일들을 저장소 루트에 업로드하거나 같은 이름 파일에 덮어씁니다.
3. Commit changes를 누릅니다.
4. 1~3분 정도 기다린 뒤 아래 주소를 새로고침합니다.
   https://inhm1234.github.io/living_forest/
5. 일반 화면에서 개발자 테스트 패널이 보이지 않는지 확인합니다.
6. 테스트가 필요하면 아래 주소로 들어갑니다.
   https://inhm1234.github.io/living_forest/?test=1
