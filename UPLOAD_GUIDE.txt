살아있는 숲 V1.70.6 test 업로드 가이드

이번 버전은 전체숲 표시와 출시 전 정리 중심 패치입니다.
Apps Script 배포가 이미 완료되어 있다면 Apps Script를 다시 배포하지 않아도 됩니다.

GitHub에 우선 업로드할 핵심 파일:
- index.html
- app.js
- style.css
- README.md
- VERSION.txt
- LAUNCH_CHECKLIST.txt
- PUBLIC_SHARE_COPY.txt
- UPLOAD_GUIDE.txt

업로드 방법:
1. ZIP 압축 풀기
2. 안의 파일과 assets 폴더를 GitHub 저장소에 덮어 업로드
3. ZIP 파일 자체는 올리지 않기
4. 업로드 후 GitHub Pages 배포가 끝날 때까지 1~3분 기다리기
5. 사이트에서 Ctrl + F5 새로고침
6. 화면 상단 버전이 V1.70.6 test로 보이는지 확인
