중요:

- assets/garden, assets/world, assets/share 폴더는 그대로 유지해야 이미지가 깨지지 않습니다.

- 방문자 판정 저장은 기존 나무 기록과 분리되어 있습니다.
- 이번 버전은 V1.7 test의 월드 숲 2.5D 화면을 세부 보정한 테스트판입니다.

GitHub 적용 순서:
1. GitHub 저장소 inhm1234/living_forest로 이동합니다.
2. ZIP 안의 파일들을 저장소 루트에 업로드하거나 같은 이름 파일에 덮어씁니다.
3. Commit changes를 누릅니다.
4. 1~3분 정도 기다린 뒤 공개 페이지를 새로고침합니다.
5. 일반 화면에서 V1.7.1 test가 보이는지 확인합니다.
6. 테스트는 공개 주소 뒤에 ?test=1을 붙여서 진행합니다.

V1.7.1 test 확인 포인트:

낮/노을/밤 테스트:
- 테스트 모드 주소 뒤에 &worldTime=day를 붙이면 낮 숲을 확인할 수 있습니다.
- 테스트 모드 주소 뒤에 &worldTime=sunset을 붙이면 노을 숲을 확인할 수 있습니다.
- 테스트 모드 주소 뒤에 &worldTime=night를 붙이면 밤 숲을 확인할 수 있습니다.
