살아있는 숲 V1.52 test 패치 업로드 가이드

중요:
- ZIP 파일 자체를 GitHub에 올리지 않습니다.
- ZIP을 압축 해제한 뒤 안의 파일만 저장소에 덮어 업로드합니다.
- 이번 버전은 GitHub 파일 업로드 + Apps Script 새 배포가 둘 다 필요합니다.

GitHub에 먼저 올릴 핵심 파일:
- index.html
- app.js
- style.css

Apps Script에 붙여넣을 파일:
- APPS_SCRIPT_CODE.txt

권장 순서:
1. GitHub에 index.html / app.js / style.css 업로드
2. Apps Script Code.gs를 APPS_SCRIPT_CODE.txt 전체 내용으로 교체
3. Ctrl+S 저장
4. 배포 > 배포 관리 > 새 버전 배포
5. 사이트 Ctrl+F5 새로고침
