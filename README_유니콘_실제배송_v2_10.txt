# 숲 유니콘 실제 배송 v2.10

## 이 패치가 하는 일
- 유니콘이 맡은 편지를 **30분 뒤** 수신자 정원에 도착시킵니다.
- 편지가 도착해도 유니콘은 **추가 30분** 동안 귀환 중입니다.
- 총 1시간 동안 유니콘은 정원에 없고, 새 편지를 맡길 수 없습니다.
- 일반 방문 동물의 `garden_letters`, 방문 상태, 기존 배송 RPC는 건드리지 않습니다.
- 특별 친구 전용 테이블/RPC로만 저장해 기존 편지 기능과 분리했습니다.

## 먼저 할 일 (Supabase)
1. Supabase Dashboard → SQL Editor
2. `SQL_SPECIAL_FOREST_FRIEND_UNICORN_V1.sql` 전체를 붙여넣고 Run
3. 오류 없이 완료되는지 확인

## 그다음 업로드할 파일
- `dev/garden-login-test/garden.js`
- `dev/garden-login-test/forest-unicorn-preview.js`

`SQL_SPECIAL_FOREST_FRIEND_UNICORN_V1.sql`은 GitHub에 올려도 되지만, **GitHub 업로드만으로 DB에 적용되지는 않습니다.** 반드시 Supabase SQL Editor에서 한 번 실행해야 합니다.

## 첫 확인 기준
1. `?forestFriendPreview=1` 주소에서 유니콘을 누른다.
2. 만남 카드 → 편지 맡기기 → 제목/내용 작성 → 맡기기.
3. 유니콘이 출발하고 사라진다.
4. 하단 편지에는 `도착까지 약 30분`이 보인다.
5. 30분 뒤 수신자 정원에서 유니콘 편지가 보인다.
6. 보내는 정원은 그 뒤에도 `유니콘이 돌아오는 중`이 30분 유지된다.
7. 총 1시간이 지나야 유니콘이 돌아온다.

## 일부러 넣지 않은 것
- 5 숲의 빛으로 유니콘을 얻는 해금 시스템
- 일반 동물 편지와 특별 친구 편지를 한 상대 기준으로 완전히 교차 차단하는 규칙
- 빠른 검수용 1분 단축 배송

이번 패치는 실제 30분/30분 규칙만 검증하는 범위입니다.
