살아있는 숲 V1.4 stable

현재 버전: V1.4 stable
이전 버전: V1.3 stable 방문자 기록 안정판
목표: V1.4.1 test에서 확인된 방문자 조건 개선판을 안정판으로 고정

이번 V1.4 stable에서 바뀐 점:
- V1.4.1 test를 V1.4 stable로 정리
- 방문자가 개인 정원에 들어가기만 해도 바로 등장하던 흐름을 줄임
- 방문자는 기본적으로 오늘 감정 기록을 완료한 뒤 확률에 따라 등장하도록 유지
- 오늘 기록 전에는 방문자 판정/자동 재생을 만들지 않음
- 이미 오늘 감정을 기록한 사용자도 방문자 판정이 아직 없으면 개인 정원 입장 시 하루 1회 방문자 판정
- 오늘 방문자 판정이 이미 있으면 새로고침해도 다시 뽑지 않음
- 나무 이름 없음, 3일 미만, 오늘 기록 전 상태에서는 방문자 안내 문구만 표시
- 하루 방문자 판정 저장 구조 유지
- 새/다람쥐 방문 애니메이션과 방문 기록 기능 유지
- README / VERSION / UPLOAD_GUIDE / LAUNCH_CHECKLIST를 안정판 기준으로 정리

유지한 것:
- 기존 저장키 livingForestV012 유지
- 테스트 저장키 livingForestV012_TEST 유지
- 방문자 판정 저장키 분리 구조 유지
- 최근 방문자 기록 3개 표시 유지
- 서버 응답 Date 헤더 우선 사용, 실패 시 기기 날짜 대체 유지
- 운세/사주/도감/운세코드 V3 계열 내용 없음
- 서버 저장, 로그인, Google 계정 기능 없음

V1.4 stable 확인 포인트:
- README.md 상단에 V1.4 stable이 보이는지
- 홈페이지 첫 화면에 V1.4 stable이 보이는지
- 개인 정원 상단에 V1.4 stable이 보이는지
- 테스트 모드 데이터 표시에 V1.4 stable이 보이는지
- 테스트 모드 “방문 조건 기록 전”을 눌렀을 때 방문자가 바로 날아오지 않는지
- 그 상태에서 감정 기록을 완료한 뒤 방문자 판정이 이루어지는지
- “방문 조건 기록 후” 버튼에서 이미 오늘 기록 완료 상태의 방문자 판정이 이루어지는지
- “방문자 새”, “방문자 다람쥐” 강제 테스트 버튼은 여전히 애니메이션과 기록 갱신이 되는지
