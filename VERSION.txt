살아있는 숲 V1.5 test

현재 버전: V1.5 test
기준 버전: V1.4 stable 방문자 조건 개선 안정판
목표: 저장 시스템을 나중에 서버 저장으로 옮기기 쉽도록 내부 구조를 정리하는 테스트판

이번 V1.5 test에서 바뀐 점:
- 기능 추가 없이 저장 데이터 구조를 정리
- dataSchemaVersion을 3으로 올림
- 저장 방식 정보를 storageInfo로 분리
- 현재 저장 방식이 local-only / browser-localStorage임을 코드 구조에 명확히 표시
- 나중에 서버 저장으로 옮길 때 사용할 자리(serverUserId, serverTreeId, lastSyncedAt)를 미리 준비
- 일반 저장키 livingForestV012 유지
- 테스트 저장키 livingForestV012_TEST 유지
- 방문자 판정 저장키 분리 구조 유지
- 기존 저장 데이터는 자동 보정되어 이어짐

유지한 것:
- 나무 이름 저장
- 하루 1번 감정 기록
- 성장 기록
- 월드 숲 내 자리 반영
- 새/다람쥐 방문자 애니메이션
- 최근 방문자 기록 3개 표시
- V1.4 방문자 조건 개선 규칙
- 기록은 아직 이 기기와 이 브라우저에만 저장됨
- 서버 저장, 로그인, Google 계정 기능 없음
- 운세/사주/도감/운세코드 V3 계열 내용 없음

저장 구조 메모:
- STORAGE_CONFIG.mode: local-only
- STORAGE_CONFIG.provider: browser-localStorage
- treeData.storageInfo: 저장 방식과 나중에 서버 이전을 위한 기본 자리
- localOwnerId: 같은 브라우저 안에서 익명 기기 소유자를 구분하기 위한 로컬 ID
- serverUserId / serverTreeId / lastSyncedAt: 아직 사용하지 않지만 추후 서버 저장을 위한 빈 자리

V1.5 test 확인 포인트:
- 홈페이지 첫 화면에 V1.5 test가 보이는지
- 개인 정원 상단에 V1.5 test가 보이는지
- 테스트 모드 데이터 표시에 V1.5 test · schema 3 · local-only가 보이는지
- 기존 나무 이름과 성장 기록이 이어지는지
- 오늘 감정 기록 잠금이 유지되는지
- 방문자 조건 개선 규칙이 유지되는지
- 방문자 새/다람쥐 강제 테스트가 정상 작동하는지
- 방문 기록 카드가 정상 작동하는지
- 구버전 보정 테스트가 정상 작동하는지
