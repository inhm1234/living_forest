살아있는 숲 V1.6.1 test

현재 버전: V1.6.1 test
기준 버전: V1.5 test 저장 시스템 구조 정리판
목표: 방문자가 다녀간 뒤 정원에 작은 흔적이 남는 느낌을 추가하는 테스트판

이번 V1.6.1 test에서 바뀐 점:
- 방문자 판정 날짜를 한국 기준 날짜 흐름에 맞게 보정
- 오늘 남은 흔적 카드에는 오늘 방문자 흔적만 표시
- 어제 방문자는 숲의 방문 기록에만 표시
- 오늘 방문자가 없으면 조용한 정원 문구 표시

V1.6 test에서 유지된 점:
- 개인 정원에 오늘 남은 흔적 카드 추가
- 작은 새가 방문하면 깃털빛 흔적 표시
- 다람쥐가 방문하면 도토리 흔적 표시
- 방문자가 없으면 조용한 정원 문구 표시
- 최근 방문자 기록에 방문자가 남긴 흔적 라벨 표시
- 기존 방문자 등장 조건은 그대로 유지
- 기존 저장 구조 schema 3 / local-only 유지

유지한 것:
- 나무 이름 저장
- 하루 1번 감정 기록
- 성장 기록
- 월드 숲 내 자리 반영
- 새/다람쥐 방문자 애니메이션
- 최근 방문자 기록 3개 표시
- V1.4 방문자 조건 개선 규칙
- V1.5 저장 구조 정리
- 일반 저장키 livingForestV012 유지
- 테스트 저장키 livingForestV012_TEST 유지
- 기록은 아직 이 기기와 이 브라우저에만 저장됨
- 서버 저장, 로그인, Google 계정 기능 없음
- 운세/사주/도감/운세코드 V3 계열 내용 없음

V1.6.1 test 확인 포인트:
- 홈페이지 첫 화면에 V1.6.1 test가 보이는지
- 개인 정원 상단에 V1.6.1 test가 보이는지
- 테스트 모드 데이터 표시에 V1.6.1 test · schema 3 · local-only가 보이는지
- 방문 조건 기록 전 상태에서 방문자가 바로 나오지 않는지
- 방문 조건 기록 후 상태에서 방문자 판정이 진행되는지
- 방문자 새 버튼을 누르면 새 애니메이션 후 깃털빛 흔적이 보이는지
- 방문자 다람쥐 버튼을 누르면 다람쥐 애니메이션 후 도토리 흔적이 보이는지
- 방문 기록 3개 버튼에서 기록과 흔적 라벨이 함께 보이는지
- 기존 나무 이름과 성장 기록이 이어지는지
- 오늘 감정 기록 잠금이 유지되는지
