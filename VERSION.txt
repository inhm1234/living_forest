살아있는 숲 V1.7.1 test

현재 버전: V1.7.1 test
기준 버전: V1.7 test — 2.5D 월드 숲 비주얼 1차
목표: 월드 숲 첫 화면의 밝기, 가독성, 내 자리 강조, 생명체 움직임을 보정하는 테스트판

이번 V1.7.1 test에서 바뀐 점:
- 월드 숲 전체 어두움과 하단 그림자 강도를 약간 완화
- 중앙 숲길과 빈터가 더 잘 보이도록 밝기와 대비 보정
- 내 나무 자리 카드 크기와 투명도를 조정해 숲과 더 자연스럽게 섞이도록 보정
- 내 나무 이름/상태 문구 가독성 개선
- 주변 자리 이름표 가독성 개선
- 새/다람쥐/나비/반딧불이/빛 입자 등장 수와 투명도 보정
- 너무 많은 생명체가 동시에 보이지 않도록 조용한 순간 확률 추가
- 테스트 모드에서 worldTime=day/sunset/night 파라미터로 낮·노을·밤 확인 가능

V1.7 test에서 유지된 점:
- 월드 숲 첫 화면 2.5D 레이어 구조
- 낮 / 노을 / 밤 분위기 기초
- 앞쪽 풀/잎/안개, 중간 숲길/빈터/내 자리, 뒤쪽 깊은 숲 레이어
- 생명체 움직임 기초 구조
- 내 나무 보기 강조 버튼

유지한 것:
- 나무 이름 저장
- 하루 1번 감정 기록
- 성장 기록
- 월드 숲 내 자리 반영
- 새/다람쥐 방문자 애니메이션
- 오늘 남은 방문자 흔적 카드
- 최근 방문자 기록 3개 표시
- V1.4 방문자 조건 개선 규칙
- V1.5 저장 구조 schema 3 / local-only
- 일반 저장키 livingForestV012 유지
- 테스트 저장키 livingForestV012_TEST 유지
- 기록은 아직 이 기기와 이 브라우저에만 저장됨
- 서버 저장, 로그인, Google 계정 기능 없음
- 운세/사주/도감/운세코드 V3 계열 내용 없음

V1.7.1 test 확인 포인트:
- 홈페이지 첫 화면에 V1.7.1 test가 보이는지
- 개인 정원 상단에 V1.7.1 test가 보이는지
- 테스트 모드 데이터 표시에 V1.7.1 test · schema 3 · local-only가 보이는지
- 월드 숲이 V1.7보다 조금 더 보기 편하고 덜 답답하게 보이는지
- 내 나무 자리 카드가 너무 크거나 답답하게 느껴지지 않는지
- 내 나무 이름과 상태 문구가 잘 읽히는지
- 주변 자리 이름표가 과하게 튀지 않으면서 읽히는지
- 새/다람쥐/나비/반딧불이/빛 입자가 너무 많거나 산만하지 않은지
- 내 나무 보기 버튼을 누르면 내 자리가 은은하게 강조되는지
- 개인 정원, 감정 기록, 방문자 흔적, 방문 기록이 기존처럼 유지되는지
- 기존 나무 이름과 성장 기록이 이어지는지
- 오늘 감정 기록 잠금이 유지되는지
