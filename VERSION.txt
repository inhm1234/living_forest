살아있는 숲 V1.8.3 test

현재 버전: V1.8.3 test
기준 버전: V1.8.2 test 낮 숲 밝기 확실 강화판
목표: 월드 숲 시간대가 내 정원에도 자연스럽게 이어지도록 만드는 테스트판

이번 V1.8.3 test에서 바뀐 점:
- 월드 숲 시간대(day / sunset / night)를 내 정원에도 연동
- 낮 숲에서 내 정원으로 들어가면 낮 정원으로 보이도록 보정
- 노을 숲에서 내 정원으로 들어가면 노을 정원 느낌으로 보정
- 밤 숲에서는 기존처럼 밤 정원 느낌 유지
- 낮 정원은 밝기, 시인성, 생기 있는 느낌 강화
- 낮 정원에서는 반딧불이 느낌을 약화하고 밝은 공기감 강화

유지한 것:
- 기존 저장키 livingForestV012 유지
- 테스트 저장키 livingForestV012_TEST 유지
- 감정 기록, 방문자 흔적, 방문 기록 유지
- 서버 저장, 로그인, Google 계정 기능 없음

V1.8.3 test 확인 포인트:
- 홈페이지 첫 화면에 V1.8.3 test가 보이는지
- 테스트 모드 데이터 표시에 V1.8.3 test · schema 3 · local-only가 보이는지
- 낮 숲에서 내 정원으로 들어가면 낮 정원처럼 더 밝게 보이는지
- 노을 / 밤도 각 시간대에 맞는 정원으로 느껴지는지
- 기존 나무 이름과 성장 기록이 이어지는지
