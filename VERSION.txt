살아있는 숲 V1.7 test

현재 버전: V1.7 test
기준 버전: V1.6.1 test 방문자 흔적 보완판
목표: 월드 숲 첫 화면을 2.5D 레이어 구조로 재정리하는 비주얼 1차 테스트판

이번 V1.7 test에서 바뀐 점:
- 월드 숲 첫 화면을 2.5D 레이어 구조로 재정리
- 숲 입구/언덕에서 깊은 숲을 바라보는 영화적 와이드 화면 방향 적용
- 뒤쪽 깊은 숲, 중간 숲길/빈터, 앞쪽 풀/안개/잎 레이어 분리
- 프론트 기준 시간에 따라 낮 / 노을 / 밤 분위기 기초 적용
- 월드 숲 안에 새, 다람쥐, 나비, 반딧불이, 빛 입자가 독립 확률로 나타나는 기초 구조 추가
- “내 나무 보기” 버튼을 추가해 월드 숲의 내 자리를 은은하게 강조

유지한 것:
- 나무 이름 저장
- 하루 1번 감정 기록
- 성장 기록
- 월드 숲 내 자리 반영
- 성장 목표 안내
- 새/다람쥐 방문자 애니메이션
- 오늘 남은 방문자 흔적 카드
- 최근 방문자 기록 3개 표시
- V1.4 방문자 조건 개선 규칙
- V1.5 저장 구조 정리
- V1.6.1 방문자 흔적 날짜 기준 보완
- 일반 저장키 livingForestV012 유지
- 테스트 저장키 livingForestV012_TEST 유지
- 기록은 아직 이 기기와 이 브라우저에만 저장됨
- 서버 저장, 로그인, Google 계정 기능 없음
- 다른 프로젝트 기능 없음

이번에 아직 넣지 않은 것:
- 서버 저장
- 실제 여러 사용자 나무 불러오기
- 완전한 3D 월드
- 고퀄리티 최종 이미지 에셋 전체 교체
- Google 로그인 기능

V1.7 test 확인 포인트:
- 홈페이지 첫 화면에 V1.7 test가 보이는지
- 개인 정원 상단에 V1.7 test가 보이는지
- 테스트 모드 데이터 표시에 V1.7 test · schema 3 · local-only가 보이는지
- 월드 숲 화면이 기존보다 더 넓고 깊은 2.5D 느낌으로 보이는지
- 앞쪽 풀/잎/안개, 중간 숲길/빈터, 뒤쪽 깊은 숲 레이어가 구분되어 보이는지
- 현재 시간에 따라 낮 숲 / 노을 숲 / 밤 숲 배지가 보이는지
- 새, 다람쥐, 나비, 반딧불이, 빛 입자가 자연스럽게 나타나는지
- “내 나무 보기” 버튼을 누르면 내 자리가 은은하게 강조되는지
- 내 나무 심으러 가기 / 오늘 내 나무 돌보기 버튼이 기존처럼 개인 정원으로 이동하는지
- 기존 나무 이름과 성장 기록이 이어지는지
- 오늘 감정 기록 잠금이 유지되는지
