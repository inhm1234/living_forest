살아있는 숲 V1.8.1 test

현재 버전: V1.8.1 test
기준 버전: V1.7.1 test 월드 숲 2.5D 세부 보정판
목표: V1.8에서 세 시간대가 모두 어둡게 느껴진 문제를 보완하고, 낮은 확실히 밝게 보이도록 시간대 대비를 강화하는 테스트판

이번 V1.8.1 test에서 바뀐 점:
- 월드 숲 시간대 분위기를 낮 / 노을 / 밤 3단계로 더 선명하게 보정
- 낮 숲은 화면 전체 밝기, 햇빛, 선명한 숲길, 빈터와 내 자리 시인성을 크게 강화
- 노을 숲은 따뜻한 빛, 긴 그림자, 하루가 끝나는 감성을 강화
- 밤 숲은 달빛, 안개, 반딧불이, 빛 입자를 더 선명하게 보정
- 월드 숲 요약 영역에 현재 시간대 분위기 안내 문구 추가
- 테스트 모드에서 낮/노을/밤을 바로 확인할 수 있는 링크 추가
- 생명체와 빛 입자 패턴이 약 20분 단위로 조금씩 달라지도록 보정

유지한 것:
- V1.7.1의 2.5D 월드 숲 레이어 구조
- 내 나무 보기 강조 효과
- 개인 정원
- 나무 이름 저장
- 하루 1번 감정 기록
- 성장 기록
- 방문자 흔적 카드
- 최근 방문자 기록 3개 표시
- 일반 저장키 livingForestV012 유지
- 테스트 저장키 livingForestV012_TEST 유지
- 저장 구조 schema 3 / local-only 유지
- 서버 저장, 로그인, Google 계정 기능 없음
- 오늘의 운세코드 관련 기능 없음

V1.8.1 test 확인 포인트:
- 홈페이지 첫 화면에 V1.8.1 test가 보이는지
- 테스트 모드 데이터 표시에 V1.8.1 test · schema 3 · local-only가 보이는지
- 낮 숲이 확실히 밝고 생명력 있게 보이는지
- 노을 숲이 따뜻하고 하루 마무리 느낌으로 보이는지
- 밤 숲이 몽환적이고 반딧불이/달빛 느낌으로 보이는지
- 내 나무 보기 버튼이 기존처럼 자연스럽게 작동하는지
- 개인 정원, 감정 기록, 방문자 흔적, 방문 기록이 그대로 유지되는지
