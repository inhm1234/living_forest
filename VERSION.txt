살아있는 숲 V1.0 체험판 공개 버전

기준 버전: V0-4.9.1 테스트용 문구 수정판
목표: 사람들에게 링크를 보내 첫 반응을 받을 수 있는 첫 공개 체험판

이번 V1.0에서 확정한 것:
- 월드 숲을 먼저 보여주는 첫 화면 유지
- 개인 정원으로 이동하는 흐름 유지
- 나무 이름 정하기 유지
- 하루 1번 감정 선택 유지
- 감정 선택 후 내 나무 성장 및 월드 숲 내 자리 반영 유지
- localStorage 저장 방식 유지
- 일반 저장키 livingForestV012 유지
- 테스트 저장키 livingForestV012_TEST 유지
- 테스트 모드 index.html?test=1 유지
- 서버 저장, 실제 로그인, Google 계정 기능은 아직 넣지 않음
- 운세/사주/도감/운세코드 V3 계열 내용은 포함하지 않음

V0-4.9.1에서 V1.0으로 바뀐 점:
- APP_CONFIG 버전을 V1.0으로 변경
- style.css/app.js 캐시 방지 쿼리를 ?v=1.0.0으로 변경
- VERSION.txt를 V1.0 체험판 공개 버전 기준으로 정리
- 기능 추가 없음
- 기존 저장키 유지

V1.0 공개 기준:
- PC 일반 화면 통과
- 첫 방문자 흐름 통과
- 이름 저장 통과
- 감정 선택 통과
- 월드 숲 반영 통과
- 테스트 모드 프리셋 통과
- 구버전 보정 통과
- 일반 화면에서 개발자 테스트 패널 미노출 확인
- 모바일 화면은 이번 점검에서 별도 캡처 확인은 생략했으므로, 공개 후 빠르게 추가 확인 권장

공개 후 다음 후보:
- V1.1: 숲의 방문자 시스템 1차
  예: 작은 새, 다람쥐, 반딧불이 등이 내 나무에 랜덤 방문하고 작은 기운/방문 기록을 남기는 구조

적용 방법:
1. GitHub 저장소 living_forest로 이동
2. index.html, style.css, app.js, VERSION.txt를 이 파일들로 덮어쓰기
3. assets 폴더는 삭제하지 말고 그대로 유지
4. GitHub Pages 반영 후 일반 주소 확인
5. 테스트 모드는 https://inhm1234.github.io/living_forest/?test=1 로 확인
