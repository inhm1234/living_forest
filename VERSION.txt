살아있는 숲 V1.2 stable

기준 버전: V1.2 test 정상 확인판
목표: 숲의 방문자 시스템 1차 안정판

이번 V1.2에서 확정한 것:
- 개인 정원에 방문자 연출 시스템 추가
- 방문자는 항상 고정으로 보이는 장식이 아니라, 찾아와서 잠시 머물고 떠나는 방식
- 방문자 1차 종류: 작은 새 / 다람쥐
- 새: 화면 밖에서 날아와 나무 근처에 머문 뒤 다시 날아감
- 다람쥐: 숲 아래쪽에서 들어와 뿌리 근처에 머문 뒤 사라짐
- 방문자 판정은 하루 1회 저장
- GitHub Pages 환경에서는 방문자 날짜 판정에 서버 응답 Date 헤더를 우선 사용
- 서버 날짜 확인에 실패하면 체험판에서는 기기 날짜로 대체
- 테스트 모드에 방문자 새 / 방문자 다람쥐 프리셋 유지

유지한 것:
- 기존 localStorage 저장키 livingForestV012 유지
- 테스트 저장키 livingForestV012_TEST 유지
- 기존 나무 성장 기록 유지
- V1.1 성장 목표 안내 유지
- 월드 숲 자리 단계 안내 유지
- 서버 저장, 실제 로그인, Google 계정 기능은 아직 넣지 않음
- 운세/사주/도감/운세코드 V3 계열 내용은 포함하지 않음

V1.2 안정 확인 완료 항목:
- 새 방문자가 찾아왔다가 머물고 떠나는 애니메이션 확인
- 다람쥐 방문자가 찾아왔다가 머물고 떠나는 애니메이션 확인
- 방문자 문구가 자연스럽다고 확인
- 기존 성장 흐름 유지

주의:
- 이번 버전은 서버 구축 전 안정판입니다.
- 날짜 판정은 GitHub Pages 서버 Date 헤더를 우선 시도하지만, 완전한 서버 검증/부정 방지는 아직 아닙니다.
- 실제 유저 전체 방문 통계나 서버 저장은 이후 서버 단계에서 진행합니다.
