살아있는 숲 V1.2.1 stable

기준 버전: V1.2 stable 방문자 시스템 정상 확인판
목표: GitHub 저장소와 홈페이지에서 현재 버전을 쉽게 확인할 수 있게 하는 버전 확인 개선판

이번 V1.2.1에서 바뀐 것:
- README.md 상단에 현재 안정판 정보를 추가
- 홈페이지 일반 화면에 V1.2.1 버전 표시 추가
- 개인 정원 상단에도 V1.2.1 버전 표시 추가
- style.css/app.js 캐시 방지 쿼리를 ?v=1.2.1로 변경
- APP_CONFIG.version을 V1.2.1로 변경

유지한 것:
- V1.2 숲의 방문자 시스템 1차 유지
- 새 방문자 애니메이션 유지
- 다람쥐 방문자 애니메이션 유지
- V1.1 성장 목표 안내 유지
- 월드 숲 자리 단계 안내 유지
- 기존 localStorage 저장키 livingForestV012 유지
- 테스트 저장키 livingForestV012_TEST 유지
- 기존 나무 성장 기록 유지
- 방문자 판정 저장은 기존 나무 기록과 분리 유지
- 서버 저장, 실제 로그인, Google 계정 기능은 아직 넣지 않음
- 운세/사주/도감/운세코드 V3 계열 내용은 포함하지 않음

V1.2.1 확인 포인트:
- GitHub 저장소 README에서 현재 안정판 V1.2.1 stable이 보이는지
- 일반 홈페이지 첫 화면에 V1.2.1이 보이는지
- 테스트 모드 데이터 표시에 V1.2.1이 보이는지
- 기존 나무 이름과 성장 기록이 유지되는지
- 방문자 새 / 방문자 다람쥐 테스트 버튼이 정상 작동하는지

주의:
- 이번 버전은 기능 추가판이 아니라 버전 확인성 개선판입니다.
- assets 폴더는 기존 것을 그대로 유지해야 합니다.
