살아있는 숲 V0-4.9 테스트용

기준 버전: V0-4.8 외부 공개 준비판
목표: 첫 공개 직전 점검과 배포 편의성 정리

변경 요약:
- APP_CONFIG 버전을 V0-4.9로 변경
- style.css/app.js에 ?v=0.4.9 캐시 방지 쿼리 적용
- 브라우저 탭/북마크용 favicon.svg 추가
- apple-touch-icon 연결
- Open Graph site_name 추가
- 모바일 브라우저 기본 감지/다크 색상 메타 보강
- JavaScript가 꺼져 있을 때 안내하는 noscript 문구 추가
- 일반 저장키 livingForestV012 유지
- 테스트 저장키 livingForestV012_TEST 유지
- 서버/로그인/Google 계정 기능은 아직 넣지 않음

확인 방법:
1. 일반 화면: index.html
2. 테스트 모드: index.html?test=1
3. 첫 화면에 개발 버전명이 노출되지 않는지 확인
4. 브라우저 탭 제목과 아이콘이 보이는지 확인
5. 이름 정하기 → 감정 기록 → 전체 숲 반영 흐름 확인
6. 테스트 모드 프리셋 6개 확인
7. 기존 기록이 유지되는지 확인
8. 모바일에서 첫 화면, 개인 정원, 하단 버튼이 잘 보이는지 확인

첫 공개 판단 기준:
- 위 8개 확인이 모두 통과하면 V1.0 체험판 공개 후보로 봐도 됩니다.
- 실제 서버 저장/로그인은 V1.0 이후 반응을 보고 붙이는 것을 권장합니다.
