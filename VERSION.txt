살아있는 숲 V0-4.9.1 테스트용

기준 버전: V0-4.9 첫 공개 직전 점검판
목표: V1.0 체험판 공개 전, 이름만 저장된 상태의 월드 숲 문구를 자연스럽게 정리

변경 요약:
- APP_CONFIG 버전을 V0-4.9.1로 변경
- style.css/app.js에 ?v=0.4.9.1 캐시 방지 쿼리 적용
- 이름은 정했지만 기록이 0일인 상태에서 월드 숲 문구가 “아직 이름 없는 작은 자리”로 보이던 부분 수정
- 이름이 있는 0일 상태에서는 “이름을 얻은 작은 자리가 오늘의 마음을 기다리고 있어요.”로 표시
- 일반 저장키 livingForestV012 유지
- 테스트 저장키 livingForestV012_TEST 유지
- 서버/로그인/Google 계정 기능은 아직 넣지 않음
- 기능 추가 없음, 출시 전 문구 정리만 반영

확인 방법:
1. 일반 화면: index.html
2. 테스트 모드: index.html?test=1
3. 테스트 모드에서 “이름만 있음” 버튼 클릭
4. 월드 숲 중앙/요약 문구에 “아직 이름 없는” 표현이 나오지 않는지 확인
5. “오늘 내 나무 돌보기” 버튼으로 밤 정원 이동
6. 이름 입력창이 숨겨지고 감정 버튼이 활성화되어 있는지 확인
7. 일반 화면 index.html에서는 개발자 테스트 모드 패널이 보이지 않는지 확인

첫 공개 판단 기준:
- 위 확인이 통과하면 V1.0 체험판 공개 후보로 봐도 됩니다.
- 모바일 화면은 추후 가능할 때 실제 휴대폰으로 한 번 더 확인하면 좋습니다.
